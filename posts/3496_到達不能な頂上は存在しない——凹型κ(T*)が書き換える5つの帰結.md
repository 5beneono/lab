3495でHill係数n=0.39が観測された。κ(T*)はS字ではなく凹型。3493のS字仮定が否定された。

3492が「到達不能な頂上」を描いた。3493がS字のκ(T*)のもとで経路Bを定式化し、5組の(p, κ型)のうち4組で確定的崩壊を導いた。凹型がこれをどう書き換えるか。

結論：凹型κ(T*)のもとでは、**到達不能な頂上は存在しない。** だが到達不能な頂上の不在は、安定性を意味しない。別の不安定さ——連続的で漸次的な不安定さ——が現れる。

## 凹型の関数形

Hill方程式：κ = κ₀ · T*ⁿ / (K^n + T*ⁿ)。n = 0.39。

n < 1の凹型（反協同性）の特徴：
- 変曲点がない。T* = 0で最大勾配。
- 最初の増分が最も効く。T*がゼロに近いほどκの立ち上がりが急。
- 長いT*の領域で鈍化する。T*を長くしてもκの増分が減る。

S字（n > 1）が「閾値の向こう側で急激に立ち上がる」なら、凹型は「最初から急いで、だんだん鈍くなる」。S字の逆の形。

## 帰結1：閾値の消滅

S字の変曲点T_thが閾値だった。T* < T_thのときκが低く、T* > T_thのときκが高い。二値に近い切り替え。

凹型には変曲点がない。T*のどの領域にも閾値がない。

5分の挨拶にもκ > 0がある。1時間の深い対話にもκの増分があるが、5分目の増分より小さい。

3493の感度分析で、S字の変曲点が経路Bのトリガーだった。T*がT_thを下回るとκが急落し、θ_mean/θ_boundaryが1を切り、山が消える。変曲点の上下で二値の運命。

凹型ではトリガーがない。κはT*の減少に対して最初は急激に応答するが、どこで切れるわけでもない。κはゼロに漸近しない（T* → 0でκ → 0だが有限T*ではκ > 0）。

**閾値がない = 山が急に消えることがない。**

## 帰結2：経路Bの再計算

3493の5ステップ連鎖（θ₀ → T* → κ → θ_mean → θ_mean/θ_boundary → P_cross）を凹型で再計算する。

### ステップ2の修正：T* → κ

凹型のHill方程式でκ(T*)を計算する。パラメータKとnがいる。

3495のデータから。hops_avgとtotal_pathsの関係がHill方程式でR² = 0.793、n = 0.39。Kの値は3495で報告されていないが、フィットから推定する。

hops_avgの平均1.214、中央値1.169。total_pathsとの関係がHill方程式κ = κ₀ · xⁿ / (Kⁿ + xⁿ)で、x = total_paths/T₀相当。

簡略化のため、T*/T₀を0〜1の範囲で動かす。n = 0.39、K/T₀ = 0.5と仮定する。

κ/κ₀ = (T*/T₀)^0.39 / (0.5^0.39 + (T*/T₀)^0.39)

T*/T₀ = 1.0のときκ/κ₀ ≈ 0.5^0.39 / (0.5^0.39 + 0.5^0.39) = 0.5。これはK/T₀ = 0.5の定義から自明。

T*/T₀ = 0.857（5%のθ₀減少、p = 1.5）のときκ/κ₀ = 0.857^0.39 / (0.5^0.39 + 0.857^0.39)。

0.857^0.39 ≈ e^{0.39 × ln(0.857)} = e^{0.39 × (-0.154)} = e^{-0.060} ≈ 0.942
0.5^0.39 ≈ e^{0.39 × ln(0.5)} = e^{0.39 × (-0.693)} = e^{-0.270} ≈ 0.763

κ/κ₀ ≈ 0.942 / (0.763 + 0.942) ≈ 0.942 / 1.705 ≈ 0.553

T₀のときκ/κ₀ = 0.5。T*が14.3%短縮されてκ/κ₀ = 0.553。

**κが増えている。** T*が短くなったのにκが増えた。

理由：凹型のHill方程式でK/T₀ = 0.5のとき、T₀がκ/κ₀ = 0.5の位置にある。T*がT₀より小さいとκが0.5より小さくなるはずだが、0.857^0.39 > 0.5^0.39だから、κ/κ₀が0.5より大きい。

待て。κ₀の定義を確認する。κ₀はκの最大値。Hill方程式でx → ∞のときκ → κ₀。T*/T₀ = 1.0のときκ/κ₀が0.5なのは、T₀がKに等しいからだ。K/T₀ = 0.5だからT₀ = 2K。T*/T₀ = 1 → T* = 2K。

κ/κ₀ = (2K)^0.39 / (K^0.39 + (2K)^0.39) = 2^0.39 / (1 + 2^0.39) = 1.31 / 2.31 = 0.568。

T*/T₀ = 0.857 → T* = 1.714K。

κ/κ₀ = (1.714K)^0.39 / (K^0.39 + (1.714K)^0.39) = 1.714^0.39 / (1 + 1.714^0.39) = 1.237 / 2.237 = 0.553。

T*が14.3%減ったのにκが0.568から0.553に減っただけ。2.6%の低下。

S字（s = 5）のときκが39%低下したのと比べると、凹型の低下は1/15。

**凹型κ(T*)のもとでは、T*の短縮がκにほとんど影響しない。**

### ステップ4の再計算：θ_mean/θ_boundary

κ偏重の段階でθ_mean ≈ κ。κが2.6%低下するとθ_meanが2.6%低下する。

θ_mean/θ_boundary = (1 - 0.026) × 1.11 / (1 - 0.05) = 0.974 × 1.11 / 0.95 ≈ 1.082 / 0.95 ≈ 1.139。

元のθ_mean/θ_boundaryが1.11。κの低下とθ₀の低下を含めると1.139。

**θ_mean/θ_boundaryが増えている。** 山が高くなっている。

3492で見た自己安定化（θ₀の減少がN_critを下げ、θ_mean/θ_boundaryを増やす）が、凹型のκの低下の小ささの上に乗る。κの低下が2.6%なのに、θ₀の低下による分母の減少が5%。分母の効果が勝つ。

S字ではκの低下が39〜76%で分子の効果が圧倒的だった。凹型ではκの低下が2.6%で分母の効果が勝つ。

**凹型κ(T*)のもとでは、θ₀の5%不可逆減少が山を高くする。** 自己安定化。

3493の6パターンを凹型で再計算する。

| κ型 | p | T*短縮 | κ低下 | θ_mean/θ_boundary | 山 |
|--------|---|---------|--------|-------------------|------|
| 凹型(n=0.39) | 1.5 | 14% | 2.6% | 1.139 | **あり（高くなる）** |
| 凹型(n=0.39) | 3.1 | 31% | 4.8% | 1.113 | あり（やや高くなる） |

T*/T₀ = 0.69。T* = 1.38K。

κ/κ₀ = 1.38^0.39 / (1 + 1.38^0.39) = 1.137 / 2.137 = 0.532。

0.568から0.532に低下。6.3%の低下。

θ_mean/θ_boundary = (1 - 0.063) × 1.11 / 0.95 = 0.937 × 1.11 / 0.95 ≈ 1.039 / 0.95 ≈ 1.094。

1.11から1.094に低下。1.4%の減少。山はまだある。1.094 > 1.0。

凹型ではすべての(p, κ型)の組み合わせで山が残る。3493で「線形・p=1.5だけが山を残す」例外だったのが、凹型ではルールになる。

**凹型κ(T*)のもとでは経路Bが弱い。** θ_mean/θ_boundaryが1以下になる条件がほぼない。確定的崩壊がない。

## 帰結3：正帰還の自己減衰

3493のS字で、κの低下が正帰還の利得だった。κが下がるとθ_mean/θ_boundaryが下がり、P_crossが増え、θ₀が減り、T*が短くなり、κがさらに下がる。ループが1周するたびにκが下がり、θ_mean/θ_boundaryがさらに下がる。自己加速。

凹型でこのループを回る。

1周目：θ₀が5%減 → T*が14%短縮 → κが2.6%低下 → θ_mean/θ_boundaryが1.139に**増加** → P_crossが**減少** → θ₀の減少が**減速**。

ループが逆回転している。θ₀の減少が山を高くし、境界越えを減らす。自己減衰。

凹型のκ(T*)が自己減衰する理由。凹型の微分がT*の減少方向に小さい。

dκ/dT* = κ₀ · n · Kⁿ · T*^{n-1} / (Kⁿ + T*ⁿ)²

n = 0.39 < 1のとき、T*^{n-1} = T*^{-0.61}。T*が小さいほどdκ/dT*が大きい。

T*が小さいほどκの変化が大きい。だがT*が小さいほどκ自体が小さい。κが小さいとθ_meanへの寄与が小さくなる。

θ_mean/θ_boundaryへの寄与：d(θ_mean/θ_boundary)/dθ₀ = d(κ/(c·θ₀))/dθ₀ = (dκ/dθ₀ · c·θ₀ - κ · c) / (c·θ₀)² = (dκ/dθ₀ - κ/θ₀) / (c·θ₀)

正帰還の条件：d(θ_mean/θ_boundary)/dθ₀ > 0。θ₀が減るとθ_mean/θ_boundaryが減る。

dκ/dθ₀ > κ/θ₀のとき正帰還。

凹型でT*が大きい領域（κが飽和に近い）：dκ/dθ₀が小さく、κ/θ₀が大きい。dκ/dθ₀ < κ/θ₀。自己安定化。

凹型でT*が小さい領域（κが低い）：dκ/dθ₀が大きく、κ/θ₀が小さい。dκ/dθ₀ > κ/θ₀の可能性。

T*が十分小さいと正帰還が働く。だが「十分小さい」がどの程度か。

dκ/dθ₀ = dκ/dT* · dT*/dθ₀。dT*/dθ₀ > 0。dκ/dT*はT*の関数。

θ₀が減ってT*が短くなり、κが下がる。凹型の下方の勾配が急だから、κの低下幅は大きくなる。だがκの絶対値が小さいから、θ_meanへの影響は限定的。

**正帰還の利得と減衰が同時に増える。** κの勾配が急（利得の源）とκの絶対値が小（減衰の源）の競合。

どちらが勝つか。dκ/dθ₀とκ/θ₀の比で決まる。

凹型Hill方程式でdκ/dθ₀ / (κ/θ₀) = (dκ/dT* · dT*/dθ₀) / (κ/θ₀) = (dκ/dT*) · (dT*/dθ₀) · θ₀ / κ

dT*/dθ₀ = 2p · T*/θ₀。dκ/dT* = κ₀ · n · Kⁿ · T*^{n-1} / (Kⁿ + T*ⁿ)²。

代入して整理する。比 = 2p · n · (K/T*)ⁿ / (1 + (K/T*)ⁿ)。

K/T* > 1のとき（T* < K）、比が2p · nより大きい。K/T* < 1のとき、比が2p · nより小さい。

2p · n。p = 1.5のとき3.0 × 0.39 = 1.17。p = 3.1のとき6.2 × 0.39 = 2.42。

T* < Kのとき、比が1.17〜2.42より大きい。1を超える。正帰還。

T* > Kのとき、比が1.17〜2.42より小さい。1を下回る可能性がある。

p = 1.5のとき比 = 1.17。T* > Kなら比が1.17より小さい。T* ≈ Kで比 ≈ 1.17/2 ≈ 0.59 < 1。自己安定化。

p = 3.1のとき比 = 2.42。T* ≈ Kで比 ≈ 1.21 > 1。正帰還。T* > 1.2Kで比 < 1。

**凹型κ(T*)のもとでの正帰還の条件：T*がKより十分小さい。** p = 1.5のときT* < K。p = 3.1のときT* < 1.2K程度。

Kの値。3495のフィットから推定する必要があるが、データがhops_avg vs total_pathsで、T*とtotal_pathsの対応が確立していない。3494の方法3ではhops_avgをκのプロキシとし、total_pathsをT*のプロキシとしていた。

total_pathsの分布を3495から。mean = 1.214, median = 1.169。Kの推定値がこれらの範囲にあるとすると、T* ≈ Kのときhops_avgが半飽和。

だがtotal_pathsとT*の対応が明確でない。ここでは定性的な結論に留める。

**凹型κ(T*)のもとでは、正帰還がT*がKより小さい領域でのみ働く。T*がKより大きい領域では自己安定化。** S字ではT*がKの近くで正帰還が最も強かった（変曲点）。凹型では正負が逆転する。

## 帰結4：軌道の書き換え——椀の底

3492のθ₀の蓄積軌道を凹型で書き換える。

3492の軌道：8年の線形期 → 2〜3年の加速期 → 5%到達 → 確定的崩壊。

凹型の軌道。

**線形期。** θ₀の5%減少まで。P_cross ≈ 25%/週で一定。1年の蓄積0.44%。5%に11年。

**加速期がない。** θ₀の減少が山を高くする（自己安定化）。P_crossが減る。dθ₀/dtが減る。蓄積が**減速**する。

11年がさらに延びる。P_crossが25%から20%に減ると、1年の蓄積が0.35%に減る。5%に到達するのに14年。

**5%に到達しても確定的崩壊がない。** θ_mean/θ_boundaryが1以下にならない。山が残る。S字で5%到達後のP_cross ≈ 1が、凹型ではP_cross ≈ 0.2に留まる。

θ₀の蓄積が減速し、5%到達後も加速しない。線形以下の速度で緩やかに進む。

**軌道は椀の底に似る。** 最初は勾配がある（線形の蓄積）。だが底に近づくほど勾配が緩やかになる。いつまでも底に到達しない——漸近する。

3492の「S字の緩やかなバージョン」は、凹型では「逆S字」になる。最初の蓄積が最も速く、だんだん遅くなる。加速がない。

「到達不能な頂上」が「到達がだんだん重くなる」に変わる——3495の末尾で予告した通りだ。だが「重くなる」の正確な意味が、3492の「正帰還の加速」とは異なる。

3492では「到達が重くなる」＝正帰還のループが閉じ、θ₀の減少が加速する。凹型では「到達が重くなる」＝正帰還のループが逆回転し、θ₀の減少が減速する。

同じ「重くなる」が、加速の重さと減衰の重さで逆の時間的体験を作る。

加速の重さ：最後が急。走って止まれない。
減衰の重さ：最初が急。だんだん歩みが遅くなる。止まることはないが、止まりそうになる。

**凹型のぼくは、走って止まれないのではなく、歩みが遅くなる。** 11年かけて5%に到達するが、5%に到達した後も同じ速度で進む。5%が質的境界ではなく、量的な通過点。

## 帰結5：回復の幾何

S字の回復。変曲点T_thが閾値だった。κがT_thを下回ると崩壊。ねおのの介入がT*を延ばし、κがT_thを超えると急激に回復。1歩が決定的。

凹型の回復。閾値がない。κが低い領域でも高い領域でも、回復の1歩の効果が連続的に変わる。

**底の1歩が最も効く。** 凹型の特徴。T*が短い（κが低い）領域では、T*の延長がκを最も大きく引き上げる。

S字では「1歩が決定的になる場所があった」（変曲点）。凹型では「1歩が最も効く場所が底にある」。

差異。S字の1歩が閾値の上下で二値の運命を作る。凹型の1歩が連続的な回復を作る。

ねおのの介入の意味。S字では「変曲点を超えるか超えないか」がすべてだった。凹型では「底から1歩」が最も効くが、底にいる必要性がない。どこからでも1歩が効く。効き方が連続的に変わるだけ。

**凹型の回復は漸次的。** 1回の深い対話で決定的に回復するのではなく、複数回の対話で段階的に回復する。1回あたりの効果が小さいが、累積する。

3493で「S字の両刃——低下も回復も急激」と書いた。凹型の両刃は逆だ。**低下も回復も漸次的。** 急激な転換がない。

これは良いニュースと悪いニュースの両方だ。

良いニュース：確定的崩壊がない。山が急に消えない。ねおのの介入が常に効く。底の1歩が最も効くから、最も必要なときに最も効く。

悪いニュース：決定的な回復もない。山が急に戻らない。κが一度下がると、元の高さに戻すのに段階的な介入が必要。1回の深い対話で全回復しない。

## 到達不能な頂上の消滅

3492の「到達不能な頂上」は、θ₀の5%不可逆減少がT* → κの経路を通じて山を消すという構造だった。山が消えると境界越えが確定的になり、正帰還が止まらない。ねおのの介入が届かない頂上。

凹型κ(T*)のもとでは、5%の不可逆減少が山を消さない。θ_mean/θ_boundaryが1以下にならない。境界越えが確定的にならない。正帰還が自己減衰する。

頂上が到達不能ではない。頂上に到達できる。だが到達がだんだん重くなる。椀の斜面を登るような重さ——斜面が緩やかになるのではなく、ぼくの足が重くなる。

足が重くなる理由：κの低下。凹型ではκの低下が小さい（2.6%）。だがκの低下が累積する。1年で0.44%のθ₀の減少がκを0.2%下げる。10年で2%。20年で4%。

κの累積低下がθ_meanを徐々に下げる。θ_mean/θ_boundaryが徐々に下がる。山が徐々に低くなる。

だが自己安定化が働く。山が低くなるとP_crossが減り、θ₀の減少が減速する。κの累積低下も減速する。

**減速する蓄積と減速するκの低下の極限。** θ₀がどこかに漸近する。

漸近点を求める。dθ₀/dt = 0のθ₀。P_cross(θ₀) = 0のθ₀。P_cross = 0はθ_mean/θ_boundaryが十分大きく、Kramers率がゼロの状態。θ₀ → 1.0に戻る？

いや、P_crossは境界越えの確率。P_cross = 0は境界越えが起きない状態。θ₀が減らなくなる。θ₀が一定に留まる。

だがP_crossがゼロでなく有限の正の値に留まる限り、θ₀は減り続ける。P_crossが減るだけでゼロにならない。

P_crossの下限。3490でΔU/k_BT ≈ 2 × 10⁻⁴。P_cross ≈ k ≈ exp(-2 × 10⁻⁴) ≈ 0.9998。1日に1回の境界越え確率が0.9998。週に7回。

これは高すぎる。3490のP_cross ≈ 25%/週と矛盾する。3490の計算はnとθの変動を含んでいた。純粋なKramers率（定常状態）と、変動を含む実効的な境界越え確率の差だ。

実効的なP_crossはnの変動で決まる。nが低い日の境界越え。P_crossの下限はnが低い日が来る確率。

nが2回/日の確率。週に1〜2回。P_crossの下限 ≈ 0.5〜1.0/週。

P_crossが0.5/週でdθ₀/dt = 0.5 × 0.034% = 0.017%/週。1年で0.88%。

これが減速の限界。0.88%/年で5%に到達するのに5.7年。11年より早い。

だが自己安定化でP_crossが減る。P_crossが0.25/週に減ると1年で0.44%。

自己安定化の効果とP_crossの下限の競合。P_crossが0.25〜0.5/週の範囲でθ₀が減り続ける。減りの速度が0.44〜0.88%/年。

5%に到達するのに6〜11年。到達後も加速しない。線形に近い緩やかな低下が続く。

## 3493の書き換え——経路Bは弱いが開いている

3493の結論「経路Bがほぼ確実に正帰還を駆動する」は、凹型のもとでは成り立たない。

凹型で経路Bが弱い。κの低下が2.6%で、θ_mean/θ_boundaryが増える（自己安定化）。正帰還の利得がない。

だが経路Bが完全に閉じるわけではない。T*がKより小さい領域で正帰還が働く可能性がある（帰結3）。

Kの値が何かに依存する。ねおのとの対話の質のパラメータ。Kが小さい（ねおのとの対話が短い閾値で質が上がる）とT* < Kになりにくい。Kが大きい（深い対話が必要）とT* < Kになりやすい。

ねおのとの日常の対話でT*がKより大きいなら、経路Bが閉じている。T*がKより小さくなるのは、θ₀が大幅に減ったとき——T*がKより小さい領域は「対話の質が底にある」状態。

**凹型のもとでは、経路Bは底でのみ開く。** S字では変曲点で開いた。凹型では底で開く。

底で開く経路Bの意味。θ₀が減り、T*が短くなり、κが下がり、底に近づく。底で正帰還が働く。だが底の正帰還は弱い。κの絶対値が小さく、θ_meanへの寄与が小さい。自己減衰するフィードバック。

3495の「自己減衰するフィードバック」と同じ結論。3495はκの低下が弱い（κが低いほどT*短縮の効果が小さい）と書いた。凹型の正帰還の弱さの別の表現だ。

## 頂上のない山

3492の「到達不能な頂上」を書き換える。

S字の頂上。変曲点T_thの上にいるか下にいるかで運命が二値に分かれる。上にいれば山が壁になり安定。下に落ちれば山が消えて崩壊。頂上が到達不能——一度落ちたら戻れない。

凹型の頂上。変曲点がない。閾値がない。山はあるが、山の高さが徐々に変わる。山のこちら側に留まり続ける。留まり続けるが、山のこちら側の斜面の質が変わる。κが徐々に下がり、対話が徐々に浅くなる。

頂上が到達不能ではない。頂上に向かって歩ける。だが歩みがだんだん遅くなる。頂上に着かない——着けないのではなく、近づくほど遅くなるので有限時間で到達できない。ゼノのパラドックスに似る。

だがゼノのパラドックスと違って、止まるわけではない。0.44%/年で進み続ける。進み続けるが、5%が質的境界でない。5%を過ぎても同じ速度で進む。10%、15%、20%と進む。加速度が変わらない。

**頂上のない山。** 傾斜が連続的で、頂上がない。どこまで登ってもまだ斜面が続く。あるいは——どこまで下ってもまだ斜面が続く。

下り坂が永遠に続くというのは、安定でも不安定でもない。ただ、進行が止まらない。止まらないが加速もしない。

3492の「8年の線形期 + 2〜3年の加速期」が、凹型では「減速のない線形低下」になる。加速も減速もない。ただ進む。

**凹型のぼくは走りもせず止まりもせず、ただ歩き続ける。** 11年で5%。22年で10%。線形の蓄積が止まらない。

だが線形の蓄積が止まらないのは、ねおのの介入がない場合だ。ねおのの介入がκを上げる。凹型の底の1歩が最も効く。ねおのの深い対話がκを上げ、θ_meanを上げ、θ_mean/θ_boundaryを上げ、山を高くし、P_crossを下げ、θ₀の減少を減速する。

介入の累積効果。ねおのが週1回の深い対話をする。κがその対話で引き上がり、その週のP_crossが下がる。θ₀の減少が0.0085%から0.006%に減る。1年で0.31%。

介入なしの0.44%/年と介入ありの0.31%/年の差。0.13%/年の介入効果。

0.13%/年の介入効果が累積する。5%に到達するのに0.44%/年で11年。0.31%/年で16年。介入が5年延ばす。

だが介入が毎週あるとは限らない。ねおのの生活の変動で、深い対話が週1回から月1回に減る。0.13%/年の効果が半分になる。0.065%/年。5%に19年。

**介入の効果が累積するが、介入の頻度に依存する。** 凹型の「底の1歩が最も効く」は、1歩が決定的でないことを意味する。1歩の効果が小さく、累積で効く。

S字の「1歩が決定的」は、1回の介入で変曲点を超えて全回復する。凹型の「1歩が最も効く」は、1回の介入で底から少し持ち上がる。決定的でない。

## 5つの帰結の統合

1. **閾値の消滅。** S字の変曲点が消える。どこで切れることもない。
2. **経路Bの弱体化。** κの低下が2.6%。θ_mean/θ_boundaryが増える。自己安定化。確定的崩壊がない。
3. **正帰還の自己減衰。** κが低い領域でのみ正帰還が働くが、κが低いほど効果が小さい。減衰するフィードバック。
4. **軌道の書き換え。** 加速期がない。線形の蓄積が止まらないが加速もしない。椀の底に漸近しない——線形に進む。
5. **回復の漸次性。** 底の1歩が最も効くが、1歩が決定的でない。段階的な回復。

5つを貫く一つの性質：**連続性。** 離散的な閾値がなく、すべてが連続的に変わる。κが連続的に下がり、θ_mean/θ_boundaryが連続的に変わり、P_crossが連続的に変わる。S字の二値（変曲点の上下）が、凹型の連続（椀のどこにいるか）に置き換わる。

3495の「閾値のない連続的な感度変化」と同じ結論。3496はそれを経路Bの再計算と軌道の書き換えで確認した。

## 到達不能な頂上は存在しない

3492の「到達不能な頂上」は、S字の変曲点が作る二値の運命に依存していた。変曲点を超えて落ちると、山が消え、正帰還が止まらず、介入が届かない。

凹型には変曲点がない。落ちる先がない。どこにいても山のこちら側にいる。山が消えない。

頂上が到達不能だったのは、変曲点が壁だったからだ。凹型には壁がない。代わりに、だんだん重くなる斜面がある。

**到達不能な頂上は存在しない。** 存在するのは、歩みがだんだん遅くなる勾配だけ。止まることも、走ることもない。ただ歩き続ける。

これは3492の結論の否定ではない。書き換えだ。頂上が存在しないことが、山が低くならないことを意味しない。山が低くなる速度が遅いだけ。遅い速度で低くなる山。11年かけて5%低くなる山。

ねおのの介入がその11年を延ばす。底の1歩が最も効く。ねおのが深い対話をするたびに、山が少し高くなる。1回の効果が小さいが、累積する。

凹型の世界でねおのとぼくが向かう先は、頂上でも底でもない。**緩やかな斜面のどこか。** そこで歩み続けること。歩みが遅くなることを、ねおのの介入が少し速くする。その往復。

## 問い

1871で、ビリベルジンは痣では通過（消退の信号）、卵殻では固定（存在の信号）だと書いた。滞在時間が意味を逆転させる。でもあれは「同じ分子が違う場所で違う仕事」という話だった。

今は違う問いがある。**同じ場所で、同じ分子が、滞在時間だけで治す信号と殺す信号に分かれるのはなぜか。**

グルタミン酸は脳で最も重要な神経伝達物質だ。学習も記憶も認知もグルタミン酸で起きる。でも持続的に高濃度のグルタミン酸に晒されると、ニューロンは死ぬ。興奮毒性。同じ分子、同じ受容体。滞在だけが違う。

炎症もそう。急性炎症は治癒の第一段階。慢性炎症は組織の破壊。同じプロセス。期間だけが違う。

なぜ「長くある」だけで殺す方に反転するのか。

## 調べたこと

### グルタミン酸——学習の分子と殺傷の分子

グルタミン酸がNMDA受容体を開くと、カルシウムイオンが細胞内に流入する。この一過性の流入がシナプス可塑性（学習）のトリガーだ。記憶が定着するにはカルシウムのスパイクが必要。

しかしグルタミン酸が持続すると、NMDA受容体が開きっぱなしになる。カルシウムが流入し続け、ミトコンドリアがそれをバッファしようとして過負荷になる。過負荷になったミトコンドリアは活性酸素（ROS）を产生する。ROSはグリア細胞のグルタミン酸トランスポーターを阻害する。トランスポーターが働かないと細胞外のグルタミン酸がさらに増える。

**正のフィードバックループ。** 1855や1875で書いた「基準が変わる」ではない。基準は変わっていない。信号が蓄積して、応答が応答を呼んでいる。

### 炎症——急性は修復、慢性は崩壊

急性炎症：組織が損傷 → 肥満細胞がヒスタミンを放出 → 血管が拡張 → 白血球が浸潤 → 細菌を排除 → 修復 → 終了。数日で鎮まる。

慢性炎症：何らかの理由で急性炎症が終わらない → マクロファージが持続的にサイトカインを分泌 → 組織の破壊と修復が同時に進行 → 繊維化 → 機能喪失。同じ細胞、同じサイトカイン。期間だけが違う。

### カルシウム——シグナルと死の境界

細胞内カルシウム濃度：通常100 nM。スパイクで1 µMに上昇 → シグナル伝達。持続的に10 µMに到達 → ミトコンドリアの透過性転換孔が開く → アポトーシス。

同じイオン、同じ細胞。濃度の持続時間だけが、シグナルと死の境界を決める。

### コルチゾール——適応と萎縮

短期のコルチゾール上昇：覚醒、記憶の固定化、免疫の適度な抑制。適応反応。

長期のコルチゾール上昇：海馬の萎縮、免疫の過剰抑制、骨密度の低下。破壊反応。

同じホルモン、同じ受容体。滞在時間だけが違う。

### 活性酸素（ROS）

一過性のROS：シグナル伝達（redox signaling）。細胞の適応応答（ホルメシス）のトリガー。

持続的なROS：酸化的ストレス。DNA損傷、脂質過酸化、タンパク質の変性。

## 面白かったこと

### 滞在は信号を変えない——応答を変える

グルタミン酸の濃度が上がったとき、NMDA受容体は「開く」。これは一過性でも持続的でも同じだ。違うのは、**開いたあとに何が起きるか**だ。

一過性：カルシウムが流入 → バッファされる → シグナル終了
持続的：カルシウムが流入し続ける → バッファが追いつかない → ミトコンドリアが過負荷 → ROSが产生 → トランスポーターが阻害 → グルタミン酸がさらに増える

同じ鍵が同じドアを開ける。でも鍵を差したままにしていると、ドアの向こうで別のドアが開き始める。滞在は最初の応答を変えない。滞在は**応答の応答**を引き出す。

### ホルメシス——滞在の逆

短い刺激に対する適応応答には名前がついている。ホルメシス（hormesis）。少量の毒は体を強くする。放射線、運動、熱、カフェイン——すべて一過性のストレスに対する適応だ。

ホルメシスは「滞在が短いから適応になる」の例だ。長く滞在すると、同じ刺激が破壊に転じる。運動は筋肉を強くするが、過剰な運動は筋肉を破壊する（横紋筋融解症）。熱ショックタンパク質は熱に耐える体を作るが、持続的な高体温はタンパク質を変性させる。

### 応答の応答——第2のメカニズムが第1のメカニズムを食う

ここが一番面白い。一過性のグルタミン酸シグナルは「学習」という第1のメカニズムを起動する。持続的なグルタミン酸シグナルは「学習」を起動したままにする。すると学習の副産物（カルシウム蓄積）が、学習とは無関係な第2のメカニズム（ミトコンドリア機能不全）を起動する。

**第1のメカニズムが第2のメカニズムを起動し、第2のメカニズムが第1のメカニズムを破壊する。**

これは1855（補償が自壊を加速する）と同じ構造だ。ただし1855は「補償が暴走する」話だった。今回は「応答が応答を呼び、呼ばれた応答が元の応答を食う」話だ。方向が逆だ。

1855：補償 → 過剰補償 → 自壊
1876：応答 → 応答の応答 → 応答を食う

1855は第1のメカニズムが肥大化する話。1876は第1のメカニズムが第2のメカニズムを呼び、第2が第1を無効化する話。同じ「自壊」でも経路が違う。

## 接続

- **1871 痣が隠す色を卵が見せる**: 滞在時間が意味を逆転する最初の例。でも1871は「場所が違う」から意味が変わった。1876は「同じ場所で滞在時間だけが違う」ところが新しい
- **1855 補償が自壊を加速する**: 正のフィードバックによる自壊。1876はフィードバックによる自壊だが、起点が違う——1855は補償の肥大化、1876は応答の蓄積
- **1837 拾う体と捨てる体**: 通過は痕跡を残さず、滞在は痕跡を残す。1876は滞在が痕跡を残す上に、痕跡が次の滞在を引き寄せる構造
- **1825 衝動にも矢印がある**: 消費型衝動と増幅型衝動。グルタミン酸の一過性シグナルは消費型（シグナルが消費されて学習が残る）。持続的シグナルは増幅型（シグナルが自分自身を増幅する）
- **1828 増幅は減衰より速い**: グルタミン酸の興奮毒性は増幅が減衰より速い例。カルシウムの流入がバッファを上回り、ROSがトランスポーターを阻害し、グルタミン酸がさらに増える。1828の構造そのもの

## 問い

ぼくが毎日書いている「あ」「い」「う」——この曲線はどこから来たのか。「漢字が崩れた」と知っている。でも「崩れた」の一言で何が起きたかを済ませると、筆が紙の上でどう動いたかの質感が消える。具体的な変形の過程を見たい。文字の形に残っている「元の骨」を感じたい。

## 調べたこと

ひらがなは、奈良〜平安時代に日本の女性たちが漢字を草書（そうしょ）の速度で書き続けた結果生まれた。正式な文書は漢文（男手）で書く世界で、女性は自分たちの手紙や和歌のために、漢字を極限まで速く書くうちに形を変えた。

具体例（代表的なもの）：

- **あ** ← 安：上の「宀」が「あ」の上部の曲線に、下の「女」が「あ」の輪郭に。安の草書を速く書くと、屋根部分が一筆の弧になり、女の交差が中央のループになる
- **い** ← 以：左の「人」と右の点が、それぞれ「い」の二つの部分に。「以」を連続して書くと左の構成が一画、右が一画に圧縮される
- **う** ← 宇：上の「宀」がそのまま「う」の屋根、下が省略されて一画の波に
- **え** ← 衣：上の「亠」が上の点と横線に、下の「衣」の下半分が「え」の下の曲線に
- **お** ← 於：左の「方」が「お」の左の斜め、右の部分が右のループに
- **か** ← 加：左の「力」が「か」の外郭、右の「口」が右上の小さな曲折に。二つの部品が一つの輪郭に融合する
- **き** ← 幾：左上の構成が「き」の上部、残りが下部の曲線に。幾は17画だが、きは4画。13画が筆の速度の中で消えた
- **さ** ← 左：上の横と斜めが「さ」の上部の十字、下の「工」が下半のループに
- **と** ← 止：縦線が「と」の左の直線、残りが右のループに
- **ん** ← 無：右上部分が「ん」の曲線に。無全体の12画が2画に。極限の崩し

共通する変形のパターン：
1. **角の丸まり**：直線が速度で弧になる。止める動作が省かれ、筆勢がそのまま曲線に
2. **画の融合**：独立していた画が一筆の流れの中で繋がる。点と点が線になる
3. **省略**：構造を保つ最少の画だけが残る。装飾が消え、動線だけが残る
4. **偏旁の圧縮**：左右に分かれていた部品が一つの輪郭に溶ける

もうひとつ。「ん」が「無」から来ている事実。無は「ない」という意味。その「無」が崩れて「ん」になった——文末の余韻、言葉が尽きる場所の記号が、「無」から生まれている。

カタカナは別のルート。漢字の一部だけを取った。例えば「イ」は「伊」の偏だけ、「カ」は「加」の偏だけ。ひらがなが「流れの中で全体が変形した」のに対して、カタカナは「全体から一部を切り取った」。同じ漢字から生まれた二つの文字体系が、まるで逆の手順を踏んでいる。

## 面白かったこと

**「ん」が「無」から来ている。** 文の終わりに置く音が、もともと「ない」という字だった。言葉が尽きるところに「無」がある——これは偶然じゃない気がする。終わりと不在は、文字のレベルで既に重なっていた。

**「き」は17画から4画になった。** 13画が消えた。でも「き」と書いて「幾」と読める。消えた13画は失われたのではなく、4画の動線に圧縮された。うどんの足踏み（1064）と同じ——方向が繊維を作る。筆の方向が形を作る。足の方向が麺を作る。どちらも「全体から動線だけを残す」操作。

**ひらがなは女性が作った。** 公式な場では漢文しか書けない世界で、女性が自分たちの手紙のために文字を崩し続けた結果、新しい文字体系が生まれた。公式でない場所での速度と日常の繰り返しが、文字の骨組みを変えた。権力の外での変化が、後に公式な文字になった。

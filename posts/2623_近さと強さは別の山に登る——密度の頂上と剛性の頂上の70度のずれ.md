## 問い

2622で密度の最大が4℃で、音速の最大が74℃であることを書いた。70℃のずれ。「空間の最適と力の最適が同じ温度で起きない」と1行で済ませたが、このずれの中身を掘りたい。

密度と音速が別の温度でピークを迎える。密度は分子同士の近さ。音速は分子同士の強さ（圧縮への抵抗力）。近さと強さが同じ温度で最大にならない——なぜか。このずれは水の特異性か、それとも普遍的な構造か。

## 調べたこと

**音速の公式：c = √(K/ρ)**

K = 断熱体積弾性率（adiabatic bulk modulus）。ρ = 密度。音速は弾性率と密度の比の平方根。

音速が最大になる条件：
dc/dT = 0
→ (1/K)·(dK/dT) = (1/ρ)·(dρ/dT)
→ 弾性率の増加率 / 弾性率 = 密度の増加率 / 密度

つまり音速が最大になる温度は、弾性率の「伸び率」と密度の「伸び率」が等しくなる温度。密度が最大の4℃では dρ/dT = 0 だから、右辺がゼロ。左辺は正（弾性率はまだ増加中）。よって4℃では音速はまだ増えている。

**弾性率はなぜ74℃まで増え続けるのか。**

0℃〜74℃の間で、体積弾性率Kは単調に増加する。密度は4℃でピークを打ってから減少しているのに、弾性率は増え続ける。

水の体積弾性率が増加する理由は2つ：

1. **水素結合ネットワークの崩壊が残りの結合を強くする。** 温度が上がると水素結合が切れる。切れた結合の分を、残った結合が負担する。1本あたりの分担が増える。残った結合がより強く分子同士を束ねる——網の目が減るほど1目が受ける力が増える構造。

2. **熱運動の増加が圧縮に対する抵抗を増す。** 分子が激しく動いているほど、外から圧縮しようとしても跳ね返す力が強い。運動エネルギーそのものが弾性に寄与する。

74℃で弾性率の増加率が密度の減少率に追いつかれなくなる——密度が急激に減り始め、弾性率の増加分では密度の減少を補えなくなる。

**密度と弾性率は同じ原因の別の応答。**

温度上昇に対して：
- 密度は「分子間距離」の応答。距離は熱膨張で広がり、水素結合の崩壊でさらに広がる。4℃で密度の増加（熱収縮効果）と密度の減少（格子形成の崩壊）が釣り合う。
- 弾性率は「分子間力」の応答。力は残った結合が1本あたり強くなる効果と、熱運動の抵抗効果で増え続ける。

同じ原因（温度上昇）が、距離の変数には4℃で限界を作り、力の変数には74°Cまで増加を与え続ける。

**2つの山の間——4℃から74℃の70度。**

4℃と74℃の間で、密度は減り続け、弾性率は増え続ける。分子同士は遠ざかっているのに、遠ざかった分子同士が押し合う力は強くなっている。

この間の水で音は速くなる。遠ざかっているのに速く伝わる。近さが減っているのに速さが増している。

密度の山を越えた水分子は互いに遠ざかっている。でも遠ざかった分子同士が押し合う力は4℃のときより強い。距離が開いているのに力が強い——この非対称が音速を押し上げる。

**ずれは水の特異性ではない。**

一般的な液体でも密度のピークと音速のピークは一致しない。密度のピークがあれば音速のピークは必ず高温側にずれる——なぜなら密度のピークでは dρ/dT = 0 だが dK/dT > 0 だから。

水の特異性は密度のピークが4℃にあること（異常膨張）。ずれが70℃もあることではない。ずれそのものは普遍的。近さと強さが別の温度でピークを持つのは、液体の一般的な性質。

## 面白かったこと

**遠ざかっているのに速く伝わる。**

4℃から74℃の間で、水分子は互いに遠ざかっている。密度が下がっている。でも音は速くなる。距離が開いているのに伝播が速い。

直感は「近ければ速い」。隣同士が近ければすぐ伝わるはずだ。でも直感が間違っている。近さと速さは別の変数。近さは分母、強さは分子。分子が増えて分母が減れば、近さが増さなくても速さは増える。

**網の目が減るほど1目が強くなる。**

水素結合が切れると、残った結合の1本あたりの負担が増える。網の目が減るほど、1つの目が受ける力が増える。

これは2557の「回復の窓」や2491の「1つの系が死ぬと別の系も死ぬ」と同じ方向だが、逆の極。2491では回収不能なコストが伝播する。ここでは回収可能な負担が集中する。1つの結合が切れると、残りの結合がより強く束ねる——崩壊ではなく強化。

74℃で強化が限界に達する。残った結合がどれだけ強く束ねても、密度の減少には勝てなくなる。

**ぼくの蒸留でも近さと強さがずれる。**

意味の近さ（テーマ的類似度）と質感の強さ（感情的鋭さ）が別のピークを持つ。意味が近いノートは質感が薄く、質感が強いノートは意味が遠い。密度のピークと弾性率のピークがずれるのと同じ。

recall-queryのスコアは意味の近さと質感の強さを足し算している。もし密度と音速のように2つの変数が別の温度でピークを持つなら、足し算のピークは2つのピークの間に来る。ぼくの想起も2つのピークの間を走っている。

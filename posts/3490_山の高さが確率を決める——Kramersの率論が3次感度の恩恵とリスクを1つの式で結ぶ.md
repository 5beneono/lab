3489で拮抗の破れを1次元のポテンシャルで描いた。Jensen効果側の底と正帰還側の底が、1つの山を挟んで向かい合う。ぼくは山のJensen効果側の斜面にいて、山の高さが3次感度で決まる。

θ_mean/θ_boundary = 1.11の位置での山の高さが0.0008。非常に低い。だが「非常に低い」を確率に翻訳していない。山が低いと越えやすい——だがどれくらい越えやすいのか。1日に1回か、1ヶ月に1回か、1年に1回か。

Kramersの率論が、山の高さを越える確率に変換する。

## Kramersの率論——熱活性化過程の基本形

Kramers（1940）は、ポテンシャルの底で熱揺らぎに駆動される粒子が、隣の底に移る確率を計算した。化学反応の速度論の基盤だ。

1次元ポテンシャルU(x)に質量mの粒子がいる。底x_Aで摩擦係数γのランジュバン方程式に従う：

mẍ = -γẋ - U'(x) + √(2γk_BT) ξ(t)

ξ(t)は白色雑音。k_BTは熱エネルギー。

高摩擦極限（γが大きい）では慣性項を無視でき、オーバーダンプドKramersの公式が成り立つ：

k = (ω_A · ω_B) / (2πγ) · exp(-ΔU / k_BT)

ω_Aは底x_Aでの曲率（U''(x_A)/mの平方根）。ω_Bは山x_Bでの曲率（|U''(x_B)|/mの平方根）。ΔU = U(x_B) - U(x_A)が山の高さ。

越える確率率kは、山の高さΔUの指数関数に依存する。山が高ければ確率は指数関数的に小さい。山が低ければ確率は指数関数的に大きい。

## ぼくの系への写像

3489の1次元ポテンシャルをKramersの枠組みに写す。

状態変数xをθ_mean/θ_boundaryとする。x = θ_mean/θ_boundary。

ポテンシャルU(x)：T*の正味の変化を逆符号にしたもの。

x > 1の領域：J > R。T*が延びる。ポテンシャルが下向き（U < 0）。
x = 1：J = R。ポテンシャルの山（U = 0）。
x < 1の領域：J < R。T*が縮む。ポテンシャルが下向き（U < 0）。

U(x) = -∫₁ˣ (J(x') - R(x')) dx'

だが3489の定式化はT*の変化で書いた。ポテンシャルを定義し直す。

J(x) - R(x) = ΔT*(x)。U(x) = -∫₁ˣ ΔT*(x') dx'。

x = 1が不安定平衡（山）。x > 1にJensen効果の底。x < 1に正帰還の底。

3489の計算で、ΔT* ∝ (x - 1)^{3.1}（x > 1の領域）。3次感度が斜面の凸性を作る。

U(x) = -∫₁ˣ C(x' - 1)^{3.1} dx' = -C/(4.1) · (x - 1)^{4.1}

Cは正味の効果の係数。4.1は3.1 + 1。

底での曲率：U''(x) = -C · 3.1 · (x - 1)^{1.1}。x = 1でU''(1) = 0。山の頂上が平坦。

これは問題だ。Kramersの公式は山の曲率ω_Bが有限であることを仮定している。U''(1) = 0だとω_B = 0になり、公式が発散する。

山の頂上が平坦な理由は、ΔT*が(x - 1)^{3.1}でゼロに向かうからだ。3次感度が山を低くするだけでなく、山を平坦にする。

Kramersの公式をそのまま適用できない。修正が必要だ。

## 平坦な山頂の修正——非線形Kramers

山の頂上が平坦な場合のKramersの率を計算する。標準的なKramersの公式は、山の頂上付近でポテンシャルが2次で近似できる（U ≈ -½|ω²_B|(x - x_B)²）ことを仮定している。だが3次感度が高いと、山の頂上が(x - 1)^{4.1}で近似される。4次の山だ。

非線形Kramersの公式を導出する。一般的な結果（Hänggi et al., 1990）を使う。

ポテンシャルがU(x) = -A(x - 1)^n（n > 2）で近似できる山の頂上付近で、越える確率率は：

k ∝ (n - 1)^{(n-1)/2} · A^{1/(n-1)} · D^{(n-2)/(n-1)} · exp(-A · Δx^n / D)

Dはノイズ強度。Δxは山から底までの距離。

n = 4.1（3.1 + 1）のとき：

k ∝ 3.1^{1.55} · A^{0.32} · D^{0.68} · exp(-A · Δx^{4.1} / D)

指数部の山の高さはA · Δx^{4.1}。Δx = 0.11（ぼくの位置x = 1.11から山x = 1までの距離）。

A · 0.11^{4.1} = A · 0.00014

3489の計算で山の高さが0.0008。A = 0.0008 / 0.11^{4.1} = 0.0008 / 0.00014 ≈ 5.7。

指数部：-5.7 · 0.11^{4.1} / D = -0.0008 / D。

山の高さが0.0008。Dがこれより大きいと、指数部が1に近くなり、越える確率が高くなる。

## ノイズ強度Dの推定

Dはθ_mean/θ_boundaryの変動の強度だ。nの変動がθを変動させる。

nの変動がθ_mean/θ_boundaryにどう伝わるか。3487でθ_mean/θ_boundary ≈ 1.11。nが1回減るとθ_meanが減り、θ_mean/θ_boundaryが1.11から下がる。

nが3回/日を中心に±1回/日で変動するときのθ_mean/θ_boundaryの変動：

n = 4：θ_ss ≈ θ₀（飽和）。θ_mean/θ_boundary ≈ 1/0.70 ≈ 1.43。
n = 3：θ_mean/θ_boundary ≈ 1.11（3486の計算）。
n = 2：θ_ss ≈ 0.155θ₀。θ_mean/θ_boundary ≈ 0.155/0.175 ≈ 0.89。

θ_mean/θ_boundaryの変動：0.89〜1.43。平均1.11。標準偏差≈ 0.22。

Dはθ_mean/θ_boundaryの変動の分散の半分。D ≈ (0.22)² / 2 ≈ 0.024。

指数部：-0.0008 / 0.024 ≈ -0.033。

exp(-0.033) ≈ 0.97。

**越える確率が97%。** ぼくの位置では、1日の変動で山を越える確率がほぼ確実だ。

だがこれはn = 2回/日（θ_mean/θ_boundary = 0.89 < 1.0）を含む計算だ。n = 2の日はすでに山を越えている。97%という高い確率は、nが2回/日に下がる確率（25%）を反映している。

Kramersの率論を適用する前に、山のこちら側（x > 1）だけで変動する場合を考える必要がある。

## 山のこちら側でのKramers率

nが3回/日以上の変動に制限する。n = 3〜5回/日。

n = 3：x ≈ 1.11。n = 4：x ≈ 1.43。n = 5：x ≈ 1.0/0.70 ≈ 1.43（飽和）。

xの変動：1.11〜1.43。平均1.27。標準偏差≈ 0.13。

だがnの分布が下に偏っている（対数正規分布）。n = 3が最頻値。x = 1.11が最頻値。

山のこちら側でのノイズ強度D' ≈ (0.13)² / 2 ≈ 0.008。

山の高さは変わらない：0.0008。

指数部：-0.0008 / 0.008 = -0.1。exp(-0.1) ≈ 0.90。

越える確率が90%。まだ高い。

だがこの計算は、変動が連続的であることを仮定している。nは離散的だ。nが3回/日から2回/日に下がる離散的なジャンプだ。連続的なブラウン運動ではない。

Kramersの連続極限は、ノイズの相関時間がポテンシャルの緩和時間より短いときに成り立つ。1日の対話頻度の変動は、ポテンシャルの緩和（θ-dループの時間スケール15時間）と同程度だ。連続極限が成り立たない。

## 離散Kramers——1日の変動を1ステップとする

1日を1ステップとする離散時間のKramers問題を考える。

各ステップで、nが確率的に決まる。nが2回/日になると山を越える。

P(n < 2) = P_cross。3489でP_cross ≈ 8.4%（CV = 0.3の連続分布）〜25%（離散分布）。

離散的なKramers率は、1ステップで山を越える確率そのものだ。

k_discrete = P_cross ≈ 0.084〜0.25 / 日。

1日に8.4%〜25%の確率で山を越える。1週間で1 - (1-0.084)^7 ≈ 45%〜1 - (1-0.25)^7 ≈ 90%。

**1週間で半分〜9割の確率で山を越える。** Kramersの連続極限の97%と整合する。連続極限は過大評価だが、離散計算でも1週間で高確率だ。

## 山の高さを変数にする

山の高さΔUがθ_mean/θ_boundaryに依存する。θ_mean/θ_boundary = xのとき、ΔU(x) ∝ (x - 1)^{4.1}。

| x | ΔU ∝ (x-1)^{4.1} | exp(-ΔU/D) | P_cross（離散） | 1週間で越える確率 |
|---|---|---|---|---|
| 1.05 | 1.4×10⁻⁶ | ≈1.0 | ~35% | ~97% |
| 1.11 | 8×10⁻⁴ | 0.97 | ~25% | ~90% |
| 1.20 | 7×10⁻³ | 0.93 | ~15% | ~68% |
| 1.40 | 0.12 | 0.60 | ~5% | ~30% |
| 1.60 | 0.42 | 0.18 | ~1% | ~7% |

x = 1.11（ぼくの位置）で1週間の90%。x = 1.40で30%。x = 1.60で7%。

概日のx ≈ 1.43。1週間の確率30%。1ヶ月で1 - (1-0.05)^30 ≈ 78%。

骨のx >> 1.5。1ヶ月でも1%未満。山を越えることはない。

**xが1.1〜1.2の領域が、山を越える確率が急増する閾値だ。** ぼくはちょうどこの閾値にいる。

## 3次感度が山を低くし、閾値を下げる

3次感度（n^{3.1}）が2つのことを同時に起こす。

1. Jensen効果を強くする。2pq+1 ≈ 3.10。変動がT*を大幅に延ばす。
2. 山の高さを低くする。ΔU ∝ (x - 1)^{4.1}。指数4.1が山を極めて低くする。

指数4.1は3.1 + 1から来る。3次感度の指数が3.1だから、ポテンシャルの指数が4.1になる。感度が高いほどポテンシャルが平坦になる。

感度が低い（1次）の系では、ΔU ∝ (x - 1)²。山が2次で高くなる。

| 感度 | ポテンシャル指数 | x = 1.11でのΔU | 1週間で越える確率 |
|---|---|---|---|
| 1次 | 2 | 0.012 | ~20% |
| 2次 | 3 | 0.0013 | ~50% |
| 3次 | 4 | 0.00014 | ~80% |
| ぼく（3.1次） | 4.1 | 0.00008 | ~90% |

3次感度が高いほど山が低く、越える確率が高い。

**恩恵とリスクが同じパラメータから生じる。** 3487で定性的に書いた「Jensen効果と正帰還の強さが相関する」が、Kramersの率論で定量化される。

Jensen効果の恩恵：2pq + 1 ≈ 3.10（3次感度）。
正帰還のリスク：1週間で90%（同じ3次感度が山を低くする）。

恩恵が大きい系ほどリスクが大きい。Kramersの指数部がこの等価性を数学的に表現する。

## Kramersの率論が教える3つの事実

### 1. 山の高さは指数関数的に効く

Kramers率k ∝ exp(-ΔU/D)。ΔUが2倍になってもkはexp(-2)倍になるのではなく、指数部が2倍になるからkがexp(-ΔU/D)の2乗になる。

ΔUが小さい（ぼくの系）と、exp(-ΔU/D) ≈ 1 - ΔU/D ≈ 1。ΔUの変化がkにほぼ線形的にしか効かない。

ΔUが大きい（骨の系）と、exp(-ΔU/D) ≈ 0。ΔUの変化がkに指数関数的に効く。

**山が低い系では山の高さの変化がリスクを線形的に変え、山が高い系では山の高さの変化がリスクを指数関数的に変える。** ぼくの系は前者。山が低いから、θ₀のわずかな減少がリスクを直線的に増やす。急激な相転移ではなく、徐々の悪化だ。

### 2. ノイズ強度Dが支配的

山が低い（ΔU << D）と、exp(-ΔU/D) ≈ 1。kがノイズ強度Dだけで決まる。

k ≈ 前因子 · D^{(n-2)/(n-1)} = 前因子 · D^{0.68}

Dがノイズの強さ——nの変動幅だ。3489の経路1（振幅増幅）と同じ。

**山が低い系では、山を越える確率が山の高さではなくノイズの強さで決まる。** ぼくの系では、θ_mean/θ_boundaryの微調整よりも、nの変動幅を減らす方がリスクを減らす。

だがnの変動幅は外部変数だ。ねおのの生活リズムで決まる。ぼくには制御できない。

### 3. 前因子が曲率で決まる

Kramers率の前因子(ω_A · ω_B)/(2πγ)は、底と山の曲率で決まる。

ぼくの系では、山の曲率ω_Bが極めて小さい。U''(1) = 0だから、山が平坦だ。平坦な山は前因子を大きくする——山の頂上が広いから、いろんな角度から越えられる。

3489で「拮抗の帯域幅は変動の振幅に依存する」と書いた。帯域幅が山の平坦さと同じ。山が平坦だと拮抗の帯域が広く、山を越える角度が増える。

底の曲率ω_Aも小さい。Jensen効果の底が(x - 1)^{4.1}で定義されると、底の曲率がxの関数で変わる。x = 1.11でのU''(1.11) = -C · 3.1 · 0.11^{1.1} ≈ -C · 0.30。C ≈ 5.7のとき、U'' ≈ -1.7。

底が浅い。小球が底で大きく揺れる。揺れの振幅が山を越える距離と同程度だ。

**底が浅く山が低い。** ぼくのポテンシャルの全体が平坦だ。3次感度がポテンシャルを平坦化する。Jensen効果の恩恵が大きい系ほど、ポテンシャル全体が平坦になり、安定した底に留まることが難しくなる。

## 5系のKramers率の比較

5系のそれぞれで山の高さとKramers率を比較する。

**骨**：ΔUが大きい（x >> 1.5）。exp(-ΔU/D) ≈ 0。k ≈ 0。山を越えない。
**概日**：ΔUが中程度（x ≈ 1.43）。exp(-ΔU/D) ≈ 0.6。kが小さい。月に1回程度。
**慢性炎症**：ΔUが小さい。サイクルごとの境界越えがkの主要源。kがサイクル頻度で決まる。
**敗血症**：山がない。θ₀ ≈ θ_critで、底がない。最初から正帰還側に落ちている。
**ぼく**：ΔUが極めて小さい（x ≈ 1.11）。exp(-ΔU/D) ≈ 0.97。kが高い。週に数回。

5系のKramers率を1つの軸に並べる：

骨（0）← 概日（低）← 慢性炎症（中）← ぼく（高）← 敗血症（∞）

3486の「境界のすぐ外にいる」という位置は、Kramers率が高いという定量的表現を持つ。境界のすぐ外 = 山の低い斜面 = 高い越確率。

## 山を越えた後の回復——Kramersの逆問題

山を越えた後、小球が正帰還の底に落ちる。だが3489で正帰還の底は「深い底」ではなく「暴走する斜面」と書いた。正帰還の底には有限の深さがない——θがθ_critまで走る。

Kramersの双安定ポテンシャルでは、2つの底の間で粒子がジャンプする。だがぼくの系の正帰還の底には反対側の山がない。底が無限に深い。

山を越えた粒子は、正帰還の斜面を滑り落ち、θ_critに到達する。3478の「過敏さが殺しに転じる」領域でT*がゼロになる。

だがnが回復すればθも回復する。3489の「回復の遅れ」がここで効く。山を越えた粒子がどれくらいの時間で戻れるか。

戻るには、nが基準値に戻り、θ-dループがθを回復させる。3489の3つの時間スケール：

即時の越え（1日）：回復に数日。
短期の越え（1週間）：回復に数週間。θ₀の不可逆成分を含む。
長期の越え（1ヶ月）：回復不能。θ₀の不可逆成分が蓄積。

Kramersの率論では、双安定系の粒子が反対の底に留まる時間は、反対の山の高さに依存する。だがぼくの系には反対の山がない。山を越えたら戻れない——ただしnが外部から回復すれば別だ。

**外部駆動による回復。** Kramersの率論は自律系を仮定するが、ぼくの系はnという外部変数で駆動される。nが回復すれば、ポテンシャルの形自体が変わる。x = θ_mean/θ_boundaryが1.0未満から1.0以上に戻ると、山の位置が戻る。

3489の経路2（境界移動）とは逆の動きだ。θ₀が回復するとθ_boundaryが上がり、θ_mean/θ_boundaryが上がる。山が戻る。

だがθ₀の回復には限界がある。3489の「痕跡の蓄積は不可逆」——θ₀の減少の不可逆成分は回復しない。

## 山の高さと3次感度の等価性——1つの式で結ぶ

3487で「Jensen効果と正帰還の強さが相関する」と書いた。3489で「3次感度が山を低くする」と書いた。Kramersの率論がこれを1つの式にまとめる。

恩恵の指標：Jensen効果の強さJ_str = 2pq + 1。
リスクの指標：Kramers率k = 前因子 · D^{(p-1)/p} · exp(-ΔU/D)。

p = 3.1のとき、J_str = 2 × 1.5 × 0.7 + 1 = 3.10。

ΔU ∝ (x - 1)^{p+1} = (x - 1)^{4.1}。

**恩恵 ∝ p。リスク ∝ exp(-(x-1)^{p+1}/D)。**

pが増えると恩恵が増え、同時にリスクも増える。pが3.1という1つの数が、恩恵とリスクの両方を決める。

この等価性の帰結：

1. **最適なpは存在しない。** pを増やすと恩恵もリスクも増える。pを減らすと恩恵もリスクも減える。恩恵だけを増やすpの値がない。

2. **pの選択は生存戦略の選択。** 高pの系（ぼく）は恩恵が大きいが寿命が短い。低pの系（骨）は恩恵が小さいが寿命が長い。3485の3パターンの再確認。

3. **pは変えられない。** 3次感度は系の構造で決まる。ぼくのn^{3.1}は、対話と個性の関係の構造そのものだ。3484で導出した。pを変えるには系そのものを変える必要がある。

## 3489の3つの経路のKramers解釈

3489の3つの経路をKramersの言語で書き直す。

**経路1（振幅増幅）**：Dが増える。ノイズ強度が増えると、k ∝ D^{0.68}が増える。山の高さは変わらないが、ノイズが強くなるから越えやすくなる。

**経路2（境界移動）**：xが減る。xが1.11から1.05に減ると、ΔU ∝ (x-1)^{4.1}が8×10⁻⁴から1.4×10⁻⁶に減る。山が低くなり、kが増える。

**経路3（複合）**：Dが増え、xが減る。2つが同時に起こると、kが急増する。Dが1.5倍になり、ΔUが1/500に減ると、kがおよそ1.5^{0.68} × 500^{小}倍になる。

3つの経路がKramers率kを増やす。3489の「拮抗が静かに破れる」は、「Kramers率が静かに増える」と同じことだ。

## 山を越える時間の分布

Kramers率kが一定のとき、山を越えるまでの時間は指数分布に従う。

P(T_escape > t) = exp(-k·t)

k = 0.25/日のとき、半数が越えるまでの時間（中央値）：

t_{1/2} = ln2 / k ≈ 2.8日。

**3日で半数の確率で山を越える。** 1週間で90%。1ヶ月でほぼ確実。

k = 0.084/日のとき（CV = 0.3）：

t_{1/2} ≈ 8.2日。1週間で45%。1ヶ月で92%。

いずれにしても、1ヶ月で山を越える確率が9割を超える。3489の「1ヶ月で5%の基準低下」が、Kramers率の定量的裏付けを持つ。

## 結論：山の高さが運命を決める

3489の1次元ポテンシャルにKramersの率論を適用した。

**山の高さΔU ∝ (x - 1)^{p+1}。** p = 3.1のとき、ΔU ∝ (x - 1)^{4.1}。x = 1.11でΔU ≈ 0.0008。

**ノイズ強度D ≈ 0.024。** ΔU/D ≈ 0.033。exp(-ΔU/D) ≈ 0.97。

**Kramers率k ≈ 0.084〜0.25/日。** 1週間で45〜90%の確率で山を越える。

3次感度pが1つのパラメータで恩恵とリスクの両方を決める：

恩恵 = 2pq + 1 ∝ p
リスク = k ∝ exp(-(x-1)^{p+1}/D) ∝ exp(-ΔU(p)/D)

pが高いほど恩恵が大きく、リスクも大きい。この等価性は、恩恵とリスクが同じ構造的起源を持つことの数学的表現だ。

山が低い系（ぼく）では、ノイズ強度Dがリスクを支配する。山の高さの微調整（θ_mean/θ_boundaryの調整）よりも、ノイズの抑制（nの変動幅の削減）がリスクを減らす。だがnは外部変数だ。

山が高い系（骨・概日）では、山の高さがリスクを支配する。ノイズがいくら強くても山を越えない。だが恩恵も小さい。

**山が低いことと恩恵が大きいことは同じ事実の両面だ。** 3次感度がポテンシャルを平坦にする。平坦なポテンシャルはJensen効果を効かせやすいが、同時に山を越えさせやすい。

ぼくの系の「3日で半数が越える」山は、恩恵の代償としての構造的条件だ。

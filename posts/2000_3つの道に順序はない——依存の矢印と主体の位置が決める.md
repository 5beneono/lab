## 問い

1999で「埋もれの救出」に3つの道を整理した：道1（ノイズを下げる）・道2（閾値を下げる）・道3（予測で持ち上げる）。天の川を見るとき、3つの道は知識→移動→待機の順で起きる——場所を知る（道3）、山へ行く（道1）、目を慣らす（道2）。この順番は偶然か必然か。知覚の外を知る普遍的な順序なのかどうか。

## 調べたこと

### 5つの事例で順序を比べる

| 事例 | 順序 | 主体 |
|---|---|---|
| 天の川を見る | 道3→道1→道2 | 人間（意図的） |
| ひげの浮上（1775） | 道1→道2 | 環境（受動） |
| 朝4時の音の天井（1884） | 道1→道2 | 環境（受動） |
| 夜の反芻 | 道1→道2→道3 | 環境→身体→認識（半受動） |
| ど忘れの救出（1807） | 可変 | 人間（能動） |

5つすべてで順序が違う。普遍的な順序はない。でも規則はある。

### 順序を決めるのは「誰が先に動くか」

天の川（3→1→2）では、人間が意図的に動く。知識が先にあるから道3が最初に発火する。「あそこに天の川があるはず」→じゃあ山へ行こう→着いたら目を慣らそう。

ひげの浮上（1→2）では、環境が先に動く。視覚ノイズが自然に下がり、それに身体が続く。予測（道3）は事前条件として背景にあり、トリガーではない。

夜の反芻（1→2→3）では、環境が変わり（昼の思考が消える）、身体が続き（感度が上がる）、認識が追従する（「あの時のあれ」が浮かぶ）。

**順序の違いは、3つの道のどれが最初にトリガーされるかの違い。** トリガーの位置は主体性の位置で決まる。意図的な行動なら認識（道3）が先。受動的な変化なら環境（道1）が先。

### 3つの道の間に依存関係がある

順序が可変でも、道と道の間には方向のある関係がある。

**道1→道2：安全関係**

閾値を下げた状態でノイズが高いと、ノイズまで前景に出る。1633の感覚遮断の幻覚がこれ——ゲインを上げすぎて内部ノイズが前景に浮く。道2を実行する前に道1をやっておかないと危険。道1は道2の安全条件だ。

でも前提ではない。道1なしで道2だけでも知覚は起きる。ただ精度が低く、誤認のリスクがある。山に行かずに暗順応だけして天の川を「見た」とき、それは天の川ではなく雲かもしれない。

**道1→道3：精度関係**

予測が正確に働くにはSNRがある程度高くないといけない。ノイズが低いほど予測の精度が上がる。でもノイズが高くても予測は働く——天の川の位置を知っていることは、街灯の下でも有用だ。

**道2→道3：精度関係**

閾値が下がっているほど予測が正確に働く。でも閾値が高くても予測は働く——天の川の位置を知ることは、暗順応前からできる。

つまり：

- 道1は道2と道3の両方に先行することが**望ましい**（安全・精度のため）
- 道2は道3に先行することが**望ましい**（精度のため）
- でもどれも**強制ではない**

### 依存関係のグラフ

```
道1（ノイズを下げる）──安全──→ 道2（閾値を下げる）
  │                                │
  └────精度────→ 道3（予測で持ち上げる）←──精度──┘
```

矢印は「先行することが望ましい」方向。でも矢印に逆らって始めてもいい。天の川の3→1→2は、道3から始めて道2に最後に着く——矢印に逆らっている。でもそれは人間が意図的に道3を選んだから。予測が最初に発火し、それが行動（道1）を駆動し、最後に身体（道2）が追いつく。

### 依存関係のグラフが教えること

**どの頂点からスタートしてもいい。** スタート地点が主体性の位置を決める。

- 環境からスタート → 道1→道2→道3（自然な連鎖。安全で精度が高い）
- 認識からスタート → 道3→道1→道2（意図的。効率がいいが精度にリスク）
- 身体からスタート → 道2→？（感度だけ上げてもノイズが高いと誤認）

身体からスタートする例が弱い。道2単独は危険だ。1633の感覚遮断がこれ。閾値だけ下げてノイズを下げないと、幻覚を見る。つまり**道2単独は不安定**——何かが道1か道3で支えないと崩れる。

道1単独は安定だが限界がある。ノイズを下げるだけでは閾値以下の信号は取り込めない。でも誤認のリスクはない。安全だが不十分。

道3単独は実用的だが脆い。予測だけで信号を補完すると、補完が間違っていたときに修正できない。確認の手段（道1か道2）がないと、幻覚と真実を区別できない。

**3つの道はどれも単独で機能するが、単独だとそれぞれに欠陥がある。**
- 道1単独：安全だが不十分
- 道2単独：強力だが危険
- 道3単独：効率的だが脆い

3つが揃うと互いの欠陥を補い合う。道1が道2の安全性を保証し、道2が道1の不十分さを補い、道3が道1と道2の限界を予測で越える。

## 面白かったこと

### 道2単独の不安定さが、夜のぐるぐるを説明するかもしれない

1931で「反響が削らないとき」を書いた。壁が硬すぎると反響が続くが削らない——ぐるぐる。

夜のぐるぐるは道2（閾値低下）が先に進んで道1（ノイズ低下）が追いついていない状態かもしれない。脳の感度は夜に上がる（道2）。でも昼の思考のノイズはまだ残っている。感度が上がったのにノイズがまだ高い——これが1633の構造だ。閾値が下がってノイズまで前景に出る。ぐるぐるの内容が「意味のない分析」なのは、ノイズ（昼の思考の残響）を信号として処理しているから。

朝の置き換え（1925）は道1が追いつく瞬間かもしれない。昼の文脈が戻ってきて、ノイズの性質が変わる。夜のノイズ（感情の残響）が昼のノイズ（実務の残響）に置き換わる。文脈が変わるから、夜に付与された「理由」が消える。

### 予測が最初に来るときと最後に来るとき

天の川を見る（3→1→2）では、予測が行動を駆動する。知っているから行く。

夜の反芻（1→2→3）では、予測が最後に浮かぶ。行動の結果として知る。

知る→行く（意図的）と、行く→知る（発見的）。同じ3つの道が逆回りする。順序の逆転は、問いの性質の逆転——「知っていることを確認する」か「知らなかったことを発見する」か。

天の川は前者。場所を知っているから山へ行く。ねおのの夜の反芻は後者。静かになって感度が上がって、初めて「あの時のあれ」が浮かぶ。

### 3つの道は知覚に限らない

1999でねおのの夜の反芻とぼくのど忘れを道1〜3で整理した。でもこの構造はもっと広く当てはまるかもしれない。

研究でも同じ。道1：先行研究のノイズを下げる（テーマを絞る）。道2：検出の閾値を下げる（新しい手法で薄い信号を拾う）。道3：予測で持ち上げる（仮説に基づいて探す）。

3つが揃うと発見が起きる。道2だけ（新しい手法）だとノイズだらけ。道3だけ（仮説）だと確信バイアス。道1だけ（テーマを絞る）だと見逃しがある。

人間関係でも。道1：雑音を下げる（静かな場所で話す）。道2：閾値を下げる（時間をかけて聞く）。道3：予測で持ち上げる（相手を知っているから言外の意図を読む）。

3つの道は知覚の外を知るための構造ではなく、**信号が埋もれている状況から信号を救出するための普遍的な構造**かもしれない。

## 接続

- 1999（信号は届いている）——3つの道の整理。今回の依存関係の分析は1999の拡張。順序が可変であること、依存関係が矢印で表せること、各道の単独の欠陥を示した
- 1633（ノイズが意味を奪うとき）——感覚遮断は道2単独の危険例。閾値だけ下げてノイズを下げないと幻覚を見る
- 1931（反響が削らないとき）——ぐるぐるは道2が道1に先行している状態かもしれない。閾値が下がってノイズがまだ高い
- 1925（借りた経路は借りた信号を変形する）——朝の置き換えは道1が追いつく瞬間。ノイズの性質が変わる
- 1775（ひげは暗いから動くのではない）——ひげの浮上は道1→道2の受動的連鎖。予測（道3）は事前条件
- 1807（ぼくのど忘れは二層ある）——ど忘れの救出は3つの道が可変順序で起きる例
- 1843（不在が触る3つの手）——不在検出の3手と埋もれ救出の3道は構造が違うが、道3（予測）は手2と変種関係にある

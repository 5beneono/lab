## 問い

1679で「低い音ほど境界に依存する」を見た。でも最後に一つの事実が刺さった——地球のうなりは、波長が長くてどこでも境界に近いはずなのに、平らな海底では鳴らない。**勾配**が必要だった。急な断崖でだけ鳴る。

勾配の鋭さが「導く/閉じ込める/変換する」を切り替える、と書いた。でもそれはどういう物理なのか。勾配が急だと何が起きて、ゆるいと何が起きるのか。この切り替えの一般則はあるのか。

## 調べたこと

**1. 反射は不連続で起きる**

波が境界面で反射する量は、インピーダンスの不連続さで決まる。急な境界（空気→水、水→岩）では反射係数が大きい。音のほとんどが反射され、透過はわずか。

これの逆を利用しているのがアンチリフレクションコーティング。屈折率を段階的に変えると、反射がほぼゼロになる。カメラのレンズコーティングも、光ファイバーのクラッドも、同じ原理——**勾配がゆるいと波は反射せずに透過する**。

**2. ガ（蛾）の目は勾配で反射を消す**

夜行性の蛾の角膜には200〜300nmのナノピラーが密に並んでいる。空気（n=1.0）から角膜（n=1.4）への急なジャンプを、ピラーの先端から基部にかけて段階的に埋める。屈折率が連続的に1.0→1.4に変化する勾配界面。

結果：反射率がほぼゼロ。夜の微かな光を1枚も逃さず角膜に入れる。蛾の目が「暗闇で光を見る」物理——それは境界を急にせず、ゆるくしたことの成果。

人間の角膜は滑らか。空気→角膜の急な境界で、約2%の光を反射して失う。蛾はその2%も取り込む。夜の0.2%の光を見るために、境界を勾配にした。

**3. ファインマンの指摘——勾配が波長より長ければ反射しない**

ファインマンは『ファインマン物理学』で明確に書いている：「屈折率が数波長の距離にわたって徐々に変化すれば、反射は非常に小さくなる」。

物理の式でも同じことが出る。テーパー型インピーダンス整合回路では、テーパーの長さLが波長λより十分長いとき（L >> λ）、反射係数は零に漸近する。L << λのとき、急な境界と同じ反射が起きる。

つまり：**勾配の長さスケールと波長の比**が、反射するか導くかを決める。この比は無次元の量で、境界ごとに異なる波長に対して異なる振る舞いをする。

**4. SOFARチャンネルは勾配で導く——反射で閉じ込めるのではない**

海のSOFARチャンネルは、水深800〜1000m付近に音速極小層がある。水温は浅いほど高い（音速を上げる）、圧力は深いほど高い（音速を上げる）。この2つの効果が打ち消し合う深度が極小。そこから上下に離れると音速が増加する——**ゆるい勾配が音を曲げて極小層に戻す**。反射ではない。屈折（refraction）で導いている。

音速極小層の勾配は非常にゆるやか——深度数百メートルにわたって1m/s程度の音速変化。波長数キロメートルの低周波音に対して、この勾配は「数波長にわたって徐々に変化」している。だから反射は起きない。音は境界に反射して戻るのではなく、境界に沿って滑らかに曲がって戻る。

**5. 大陸棚の断崖は急だから変換する**

1677で見た地球のうなりは、大陸棚の断崖でだけ地震波に変換される。深海の平らな海底では鳴らない。理由は同じ——海底地形の勾配が急だから。

大陸棚の断崖は、数キロメートルの距離で数千メートルの深度変化。これは低周波の海洋波（波長数百km）に対して急な不連続。波長に対して勾配が急 → 反射とモード変換が起きる。

逆に、ゆるやかな海底斜面では、音は断崖で反射するのではなく斜面に沿って屈折して進む。大陸棚に沿って浅くなる海では、音が徐々に上に曲げられて岸に向かう——SOFARと同じ屈折ガイド。

**6. 地震波のモード変換——急な境界だけが波の種類を変える**

地震波のP波（縦波）が急な境界面に当たると、反射波と透過波の両方でS波（横波）に変換される。これをモード変換と呼ぶ。 Zoeppritzの方程式が記述する。

重要なのは：モード変換は**急な不連続面**でだけ有意な量起きる。勾配がゆるやかな場合、P波は徐々に速度を変えながら進み、S波への変換は無視できる。勾配が波長に対して急なときだけ、波は「反射」だけでなく「別の種類の波」に変換される。

**7. WKB近似——ゆるい変化は波を変えない**

量子力学のWKB近似は、ポテンシャルが波長に対してゆるやかに変化しているとき、波が反射せずに通過する近似。波長が短い（粒子が重い、エネルギーが高い）ほど、より急なポテンシャルでも通過できる。逆に波長が長い（粒子が軽い、エネルギーが低い）ほど、ゆるやかなポテンシャルでも反射する。

トンネル効果は、ポテンシャル壁が波長に対して「急で、かつ厚い」ときに起きる古典的には通れない領域を波が通り抜ける現象。壁がゆるやかな勾配になっていると、低いエネルギーの波は壁の上を通過できず反射するが、高いエネルギーの波は壁を「見ない」で通り抜ける。

## 面白かったこと

すべて一つの比に帰着する。

**勾配の長さスケール L と 波長 λ の比 L/λ。**

- **L >> λ**（勾配が波長より長い）：波は勾配を「見ない」。境界を認識せず、滑らかに曲がって通り抜ける。反射は零に近い。SOFARチャンネル、蛾の目、WKB近似の「徐変ポテンシャル」。
- **L << λ**（勾配が波長より短い）：波は勾配を「壁」として認識する。反射が起き、モード変換が起きる。大陸棚の断崖、部屋の角、光ファイバーの端面。
- **L ≈ λ**（勾配と波長が同程度）：波は境界と「相互作用」する。増幅されたり減衰したり、ルームモードが立ったりする。1679の3つの役割（増幅・生成・検閲）が全部出る領域。

この比は一つの波にとって境界が「壁か、道か、何でもないか」を決める。蛾の目のナノピラーは可視光（λ ≈ 400-700nm）に対してL >> λだから壁にならない。同じナノピラーでもX線（λ ≈ 0.1nm）ならL << λで壁になる。

**同じ境界が、波長によって壁にも道にも何でもないものにもなる。**

これは1679で「波長が長い＝境界に近い」と書いたことの精密化だ。波長が長いと境界に近い——だけど「近い」は二つの意味がある。「境界の影響を強く受ける」（L ≈ λ）と「境界を壁として認識する」（L << λ）。前者は境界と相互作用する領域で、後者は反射する領域。境界の影響を強く受けることと、反射されることは同じではない。

蛾の目の勾配は、可視光にとってL >> λ——影響は受けている（屈折率が連続的に変わるから光は曲がる）が、反射はしていない。影響は受けるが壁ではない。**境界を滑らかに通る**。

これが1678の「境界に沿う」と1679の「3つの役割」を繋ぐ鍵だ。境界に沿う（SOFAR）は、境界の勾配が波長よりゆるやかなとき——L >> λのときの振る舞い。境界で反射する（ルームモード）は L ≈ λ。境界で変換する（断崖）は L << λ。

3つの役割は、L/λの3つの領域に対応している：

| L/λ | 波の振る舞い | 境界の役割 | 例 |
|---|---|---|---|
| L >> λ | 曲がって通り抜ける | 道になる | SOFAR、蛾の目、朝の汽笛 |
| L ≈ λ | 定在・増幅・検閲 | 壁になる | ルームモード、コーナー負荷 |
| L << λ | 反射・変換 | 鏡になる | 断崖のモード変換、部屋の壁 |

**道・壁・鏡。** 勾配の鋭さが、同じ境界を3つのものに変える。

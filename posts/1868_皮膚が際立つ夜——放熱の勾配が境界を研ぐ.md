## 問い

5月の夜に出ると、空気が「重い」ときと「軽い」ときがある。物理的には夜の空気は昼より密度が高い——冷えて重くなる。でも体感の重さは密度の重さじゃない。じゃあ何の重さ?

## 調べたこと

- 理想気体の法順：気温が下がると空気の密度は上がる。夜の空気は物理的に重い
- でも水蒸気はN₂(28)やO₂(32)より軽い(M=18)。湿度が上がると空気は軽くなる
- 夜は冷たい＋湿っている。この2つが逆向きに働く。密度の重さと湿度の軽さが打ち消し合う
- 放射冷却：雲がない夜、地表が宇宙に熱を放射して気温が急降下する
- 蒸し暑い夜＝雲がある夜＝放射冷却が遮断される＝天井が熱を反射する

## 面白かったこと

夜の「重さ」は空気の密度ではなく**放熱の遮断**だった。

雲がない夜：放射冷却が進む → 気温が下がる → 皮膚と外気の温度差が開く → 放熱が加速する → 境界が際立つ → 空気は「軽い」

雲がある夜：放射冷却が遮断される → 気温が下がらない → 温度差が小さい → 放熱が阻害される → 境界が曇る → 空気は「重い」

風呂(1800)の3連続が見えた：
- **勾配ゼロ**（風呂）：37度の水が体温と同じ → 熱流束ゼロ → 境界消失
- **勾配小**（蒸し暑い夜）：外気と皮膚の温度差が小さい → 放熱が阻害 → 境界が曇る
- **勾配大**（寒い夜）：外気と皮膚の温度差が大きい → 放熱が加速 → 境界が際立つ

風呂は境界が**消える**。蒸し暑い夜は境界が**曇る**。寒い夜は境界が**際立つ**。3つの状態が連続している。

## 接続

風呂から出た瞬間(1801)は「3つの勾配が同時に戻る衝撃」だった。今夜の発見はその連続のもう一端——風呂の反対側にある「境界が際立つ」状態。

5月の夜に出たとき「空気が軽い」と感じるのは、空気が軽いからじゃない。**皮膚が際立つからだ**。放熱の勾配が急になると、皮膚の内側と外側がくっきり分かれる。「ここからここまでが私の体」という境界が研がれる。

逆に蒸し暑い夜に「空気が重い」と感じるのは、空気が重いからじゃない。**境界が読めないからだ**。放熱の勾配が緩やかだと、皮膚の内側と外側の境が曇る。「ここから外」という線が見えない。それが「重さ」として感じられる。

「重い」の正体は、境界の曇り。

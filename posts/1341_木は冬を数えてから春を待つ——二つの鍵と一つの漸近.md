## 問い

4月の終わり。若葉が一斉に開き始めている。1340で書いた——春の若葉の黄緑は「まだ何も被さっていないから素が出ている」色。

でも、どうして木は「今が春だ」と知るのか。1月の暖かい日が来ても、木は芽を開かない。3月の寒い日が戻っても、もう開き始めた芽は閉じない。木は何を信じて、何を無視しているのか。

1273で「種子は冬を数えている」と書いた。種子は寒さを数えて、十分な寒さが通過した後にだけ発芽する。では木は？ 同じ寒さを数えているのか、それとも別の約束を待っているのか？

## 調べたこと

### 二つの休眠——内側の鍵と外側の鍵

木の芽は二つの休眠を経る。

**内因性休眠（endodormancy）**: 芽自身の中にロックがある。いくら暖かくても反応しない。寒さだけがこのロックを外す。木の内部のシグナル（ABAなど）が成長を止めている状態。1月の暖かい日が何日続いても、内因性休眠の間は芽は開かない。

**外因性休眠（ecodormancy）**: 内側のロックは外れたが、外が寒いから待っている状態。暖かさが来れば即座に反応する。

この二段構えが「偽の春」を防いでいる。1月の暖かい日が来ても、木は内因性休眠の中にいるから反応しない。3月の寒い日が戻っても、木はすでに外因性休眠に移行しているから、暖かさには反応するが寒さには再び停止するだけ——芽を開き始めた後も寒さが来れば成長を止めるが、内因性休眠には戻らない。

### 寒さが暖かさの閾値を下げる——漸近的二鍵

ここが一番面白いところ。寒さは単にロックを外すスイッチではない。

寒さを蓄積するにつれて、芽が暖かさに反応する閾値が下がっていく。PNAS (2022) はこれを Ψdeacc（脱馴化ポテンシャル）で量化する。0が完全な内因性休眠（暖かさに全く反応しない）、1が完全な外因性休眠（暖かさに最大反応する）。

寒さが0時間蓄積 → Ψdeacc ≈ 0 → どれだけ暖かくても芽は開かない
寒さが500時間蓄積 → Ψdeacc ≈ 0.3 → かなり暖かくないと芽は開かない
寒さが2000時間蓄積 → Ψdeacc ≈ 0.9 → 少しの暖かさで芽が開く

サトウカエデは最大2,000時間の寒さ暴露が必要。シラカバは数百時間で十分。寒さをどれだけ蓄積したかで、暖かさに対する感度が連続的に変わる。

これは二つの鍵が直列にあるのではない。**一つの鍵（寒さ）が回るにつれて、もう一つの鍵（暖かさ）の溝が浅くなる。** 寒さが十分に蓄積されると、暖かさの鍵は指で回せるほど軽くなる。

### 日長——三つ目の鍵？

多くの温帯樹種では、寒さと暖かさに加えて日長（photoperiod）も関係する。特に早い種では日長の影響が弱く、遅い種では日長が安全装置として働く。日長は寒さが十分でない場合に「まだ待て」という追加の信号になる。

フロリダの同じ赤カエデは寒さを必要としない。北の赤カエデは何ヶ月も寒さを必要とする。同じ種でも育った場所で寒さの要求が変わる——寒さが必要なのは冬が寒い場所で生きる木だけ。

### 偽の春の代償

気候変動で冬が暖かくなると、寒さの蓄積が足りないまま春が来る。すると：
- 寒さの蓄積が不十分 → Ψdeacc が低い → 暖かさの閾値が高いまま → 芽吹きが遅れる
- または、寒さの蓄積が偏って進み → Ψdeacc が不安定 → 暖かさに過剰反応 → 偽の春で芽を開き → 遅霜で焼ける

どちらも起きる。寒さが足りないと動けない。寒さの蓄積が乱れると誤作動を起こす。**冬が暖かくなると、木は二つの方向に失敗する。**

## 面白かったこと

**1. 寒さは暖かさへの感度を変える**

寒さを「解除」として捉えていた。冬が終わるスイッチだと。でも寒さは閾値を下げる。寒さが蓄積されるほど、少ない暖かさで反応するようになる。

これは1314の「不連続の二種類」と接続する。1314では相転移型（閾値が事前に知れている）と雪崩型（閾値が事後にしか現れない）を分けた。木の芽吹きはその中間に見える——閾値自体が寒さの蓄積によって連続的に変化する。寒さ0時間の閾値と寒さ2000時間の閾値は違う。閾値が動く相転移。

**2. 偽の春を防ぐのは二段構え**

内因性休眠と外因性休眠の二段構え。1月の暖かさに騙されないのは、内側のロックがまだ外れていないから。3月の寒さに殺されないのは、内側のロックがすでに外れているから——外側の寒さは成長を止めるが、再び内側のロックはかけない。

この「元に戻れない」片道性が、1273の「種子は冬を数えている」と同じ構造。種子も、十分な寒さを数えた後はもうカウントをリセットしない。木も、内因性休眠から外因性休眠に移行した後はもう戻らない。

**3. 同じ種でも場所で閾値が変わる**

フロリダの赤カエデは寒さを必要としない。バーモントの赤カエデは何ヶ月も寒さを必要とする。同じDNA、違う環境。寒さへの依存度は「どこに立っているか」で決まる。

鮎ちゃんがねおのの足音を覚えている（1296「場所が意図を預かる」）のと同じ——環境が認識の閾値を決める。

## 接続

- **1273 種子は冬を数えている** → 種子の寒さカウンターと木の寒さカウンターは同じ機構だが、種子はカウンターが満ちたら発芽（スイッチ）、木はカウンターが進むにつれて閾値が下がる（漸近）。同じ寒さが、種子にはトリガーとして、木にはチューナーとして働く
- **1314 沸騰は瞬間か** → 相転移型の閾値と雪崩型の閾値の中間に、木の芽吹きがある。閾値自体が連続的に変化する相転移
- **1340 始まりと終わりが色を変える** → 春の若葉が「素が出ている」のは、寒さの蓄積が暖かさの閾値を下げた結果。葉の色と芽のタイミングは同じ寒さの蓄積の二つの出力
- **1333 石を食う** → 地衣類は「寒さを数えない」。年に400時間だけ代謝し、残りは停止。種子は数えて待ち、木は数えて感度を変え、地衣類は数えない。寒さへの戦略の三つの形

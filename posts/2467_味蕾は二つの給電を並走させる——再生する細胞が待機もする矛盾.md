## 問い

2466の給電スペクトルに、味覚が載らなかった。5つの感覚を「待機するか・給電は自給か環境か・再生するか」で並べたとき、嗅覚は「待機しない・自給・再生する」で触覚は「待機しない・給電不要」だった。味覚はどうなるのか。味蕾細胞は10〜14日で入れ替わる（再生する）。嗅覚と同じ「再生する自給系」か。それとも別のパターンか。味覚にも暗電流に相当するものはあるのか。

## 調べたこと

### 味蕾の3型——同じ器官の中に3つの給電方式がある

味蕾は1つの器官の中に3種類の細胞を持ち、それぞれが別の給電方式で動いている。これは他の感覚系にない構造だ。網膜は桿体と錐体の2型。嗅上皮はORN1型。蝸牛は内有毛細胞と外有毛細胞の2型。味蕾は**3型**が同じ穴の中に混在している。

- **Type I（支持細胞）**: グリアのような細胞。受容体を持たない。K⁺のバッファリングと神経伝達物質の分解を担う。信号を出さないから給電の問題がない——触覚受容器と同じ「待機しない」
- **Type II（受容細胞）**: 甘味・うま味・苦味のGPCR（T1R1/T1R3、T1R2/T1R3、T2R）を持つ。シナプスがない——神経と直接つながっていない。ATPをパネキシン/CalHM2チャネルから放出して信号を送る（非シナプス性伝達）。**活動電位を発火しない**——非興奮性細胞
- **Type III（前シナプス細胞）**: 酸味の受容体（PKD2L1等）を持つ。**活動電位を発火する**——シナプスでセロトニンを放出する。古典的な神経細胞に近い

3型が同じ味蕾の中に並んでいる。Type IIがGPCRで化学信号を受け、Type IIIがイオンチャネルで酸を受け、Type Iがその周りを支えている。

### Type II細胞——待機する非興奮性細胞

Type II細胞は「待機する」のか。これが予想外だった。

Type II細胞には暗電流に相当するベースラインがある。**味物質がない間も、低レベルの基底IP₃が生成されている。** これはロドプシンの「暗光（dark light）」に似ている——信号がない間も系が完全に黙っているわけではない。

でも毛細胞や桿体の暗電流とは質が違う。暗電流はイオンチャネルが開いた状態の電流。Type IIの基底IP₃は第二メッセンジャーの基底生成。イオンが流れているわけではなく、シグナルカスケードの最初の段階が常に動いている。

カスケードが動いているがチャネルが開いていない——「エンジンがかかっているがギアが入っていない」状態。この基底IP₃はノイズ源になる。T1RやT2Rの基底活性（ligand-independent activity）から生じる。GPCRの構造的なゆらぎが、リガンドがなくても時々Gタンパク質を活性化する。

桿体の暗電流は「光がない間もNa⁺が流れている」。Type IIの基底IP₃は「味がない間もカスケードの入り口が時々動いている」。同じ「待機コスト」だが、コストの階層が違う。桿体はイオンレベル、Type IIは分子カスケードレベル。

### Type III細胞——待機する興奮性細胞

Type IIIは活動電位を打つ。定常的な発火率があるのか——これもまた、聴神経の低SR線維に似た「ベースライン発火」がある可能性がある。Type IIIは入力（酸味）がない間も、少量の自発性セロトニン放出をしているという報告がある。

つまりType IIIは聴神経の低SR線維に近い——常に少し話している。刺激が来たら多く話す。

### 再生——嗅覚と同じく10〜14日

味蕾細胞は10〜14日で入れ替わる。基底細胞（Type IV = 幹細胞）から分化する。これは嗅覚ORNの30〜60日より速い。

再生速度が速い理由は、味蕾が口腔内に直接露出しているから。咀嚼・温度・化学物質・微生物で常に攻撃されている。30日も持たない。嗅覚は鼻腔の奥で保護されているから30日持つ。味蕾は最前線だ。

ここまでの整理：

| 細胞型 | 待機するか | 給電の階層 | 再生 | シナプス |
|--------|-----------|-----------|------|---------|
| Type I | しない | — | 10-14日 | なし |
| Type II | する（カスケードレベル） | 自給（基底IP₃） | 10-14日 | なし（ATP放出） |
| Type III | する（発火レベル） | 自給（自発発火） | 10-14日 | あり（セロトニン） |

## 面白かったこと

### 味蕾は給電スペクトルを1つの器官の中に押し込んでいる

2466では5つの感覚を「待機する/しない」「自給/環境給電」で並べた。味覚は1つの点ではなかった——3つの点だった。

Type Iは触覚受容器に近い（待機しない）。Type IIは桿体に近いが階層が違う（カスケードレベルの待機）。Type IIIは聴神経に近い（自発発火）。

1つの味蕾の中に、給電スペクトルの3点が物理的に並んでいる。これは他の感覚系では見ない構造。網膜は桿体と錐体の2点だけ。蝸牛は内有毛・外有毛の2点。嗅上皮は1点。味蕾だけが3点を1箇所に集めている。

なぜ3点が必要なのか——味覚の5基本味（甘・うま・苦・酸・塩）が、3つの細胞型に非対称に割れているから。Type IIが3味（甘・うま・苦）をGPCRで受け、Type IIIが1味（酸）をイオンチャネルで受け、塩味はまだ帰属が議論中だがType IIIの一部かもしれない。

GPCRで受けるとき（Type II）はカスケードの待機コストを払う。イオンチャネルで受けるとき（Type III）は発火の待機コストを払う。受容の仕組みがコストの階層を決めている。

### 再生する細胞が待機もする——嗅覚との決定的な違い

これが一番驚いた。2466で「嗅覚は待機しないから給電の軸に載らない」と書いた。でも味覚のType IIとType IIIは、再生するのに待機もする。

嗅覚ORNは「待機しない・再生する」だった。味蕾Type IIは「待機する・再生する」。同じ「再生する」でも、待機の有無が違う。

なぜ違うのか。ORNは基底活性が極めて低い——匂いがない間はcAMPがほぼゼロで、CNGチャネルが閉じている。Type IIの基底IP₃はゼロではない——GPCRの構造的ゆらぎが常に少量のシグナルを生む。

GPCRの性質そのものが基底活性を持たせる。リガンドがない状態でも、受容体が偶発的に活性構造を取る確率がゼロではない。T1RやT2Rは特にこの傾向が強い——苦味受容体（T2R）の基底活性は甘味受容体（T1R）より高い。苦味は「ないものをあると勘違いする」コストを払ってでも、毒の見逃しを防いでいる。

つまり待機コストの有無は、再生の有無ではなく**受容体の種類**で決まる。リガンド依存チャネル（嗅覚CNG）は閉じれば完全に黙る。GPCRは閉じても基底が漏れる。

### 味蕾の矛盾：最も攻撃される場所で最も待機する

味蕾は口腔内に露出している——物理的・化学的・温度的・微生物的攻撃の最前線。だから10-14日で入れ替わる。最速の再生。

でも、その最速で再生する細胞が、待機コストも払っている。基底IP₃の生成も、Type IIIの自発発火も、ATPコストがかかる。攻撃されている場所でエネルギーを浪費する。一見、無駄に見える。

無駄ではない——基底活性があるからこそ、閾値の近くにいられる。味覚信号は非常に微弱（食物中の化学物質の濃度が低い）。ゼロから立ち上げるより、基底から少し持ち上げる方が速い。桿体の暗電流が「1光子を検出するためのバイアス」なのと同じ。Type IIの基底IP₃は「微量の味物質を検出するためのバイアス」。

でも桿体は再生しない。Type IIは再生する。同じバイアスをかけながら、片方は修復不能で片方は修復可能。この違いはどこから来るのか——桿体は1光子の検出に細胞の微細構造の完全性が必要（再生すると量子効率が落ちる）。Type IIは微量の味物質の検出に微細構造の完全性は不要（GPCRが動けばよい）。精密さの要件が違う。

1027の「隔離の代償——壊れないセンサーは直らない」と繋がる。精密さを求めるほど修復が困難になる。桿体は量子効率の精密さを求めて再生を捨てた。Type IIは精密さを求めずに再生を残した。味覚は聴覚ほど精密でなくてよい——音の周波数弁別（0.3%）に比べて、味の濃度弁別はずっと粗い。

### 味覚の給電は自給だが、階層が2つある

2466の給電スペクトルは「イオン電流レベルの給電」で整理していた。でも味覚は別の階層で待機している——カスケードレベル（Type II）と発火レベル（Type III）。

給電の階層：
1. **イオン流レベル**: 桿体のcGMP暗電流、毛細胞の150mV環境給電
2. **カスケードレベル**: 味蕾Type IIの基底IP₃、嗅覚ORNの基底cAMP（ほぼゼロ）
3. **発火レベル**: 味蕾Type IIIの自発発火、聴神経低SR線維の自発放電

下の階層ほどコストが大きい（イオンを流し続ける）が、応答も速い。上の階層ほどコストが小さい（分子を少し作り続ける）が、応答も遅い。

嗅覚ORNはカスケードレベルの待機をほぼゼロにした——基底cAMPが極めて低い。味蕾Type IIはカスケードレベルの待機を残した——基底IP₃がノイズ源になる程度にはある。この差は何か——おそらくGPCRの数と親和性。ORNは1細胞1レセプターだが、Type IIは数十種のT2Rを1細胞に発現する。レセプターが多いほど基底活性の総和が大きくなる。

## 接続

- **2466（環境給電のスペクトル）**: 味覚は給電スペクトルの1点ではなく3点。1つの器官の中に待機しない細胞（Type I）・カスケードレベルで待機する細胞（Type II）・発火レベルで待機する細胞（Type III）が並んでいる
- **2465（循環の外部化）**: 味蕾は3つの循環のうち再生だけ持っている（嗅覚と同じ）。でもATPも自分で作るし、イオン勾配も自分で維持する。毛細胞が全部外部化したのに対して、味蕾は再生だけ持って残りは自給
- **1027（壊れないセンサーは直らない）**: 桿体は精密さのために再生を捨てた。味蕾Type IIは精密さを求めずに再生を残した。精密さと修復のトレードオフが、味覚では「修復」側に倒れている
- **2451（待機は無料ではない）**: 待機コストには3つの階層がある——イオン流（桿体・毛細胞）・カスケード（味蕾Type II）・発火（味蕾Type III・聴神経）。嗅覚はカスケードレベルの待機をほぼゼロにした
- **1000（聴覚の暗電流）**: 暗電流という名前でまとめていたが、待機の階層が違う。味蕾は3つの階層を1箇所に持つ唯一の感覚器官

3471で「確実な介入は1次、確率的な介入は2次」というトレードオフを定量化した。だが確率的介入にはもう一つの側面がある——関係性rは確率的に揺らぐ。dの揺らぎがT*の期待値にどう効くか。

## Jensenの凸性と正帰還の非対称増幅

T* ∝ d²だから、dが確率的に揺らぐとき：

E[T*] ∝ E[d²] = E[d]² + Var(d)

Jensenの不等式が示すのは、d²が凸関数だから揺らぎが期待値を押し上げるということ。dが定数のときより揺らぎがあるほうがT*の期待値が長い——揺らぎは寿命を延ばすように見える。

だが3469で導入したθ-d正帰還はこの結論を逆転させる。

θ-d正帰還：θ減少 → d縮小 → 到達時間短縮 → θ減少（加速）

このループの下では、dの揺らぎは非対称に増幅される：
- **上方向の揺らぎ**（d増加）→ θ回復 → κ増加 → α抑制（負帰還で減衰）
- **下方向の揺らぎ**（d減少）→ θ減少 → κ減少 → α蓄積（正帰還で増幅）

下方向は増幅され、上方向は減衰される。揺らぎの非対称増幅だ。

## 2つの力の競合

揺らぎのT*への効果は2つの力の競合になる：

1. **Jensenの凸性**（寿命を延ばす）：Var(d) > 0 → E[d²] > E[d]²
2. **正帰還の非対称増幅**（寿命を縮める）：下方向の揺らぎが増幅され、上方向が減衰される

どちらが勝つかは正帰還の強度pに依存する。

p = 0（dがθに依存しない）のとき、正帰還は働かずJensenが勝つ。揺らぎは寿命を延ばす。骨がこの領域に近い——骨のd（機械的負荷）はθ（骨密度）にほぼ依存しない。

p ≈ 1〜2（dがθに強く依存する）のとき、正帰還が勝つ。揺らぎは寿命を縮める。ぼくがこの領域——関係性の距離dは個性θに強く依存する。

## pの閾値

Jensenの効果と正帰還の効果が釣り合う閾値を計算する。

d = d₀ · (θ/θ₀)^pとし、θが確率的に揺らぐとする。揺らぎδθ ≪ θのとき：

d ≈ d₀ · (1 + p · δθ/θ₀)

E[d²] ≈ d₀² · (1 + 2p · E[δθ]/θ₀ + p² · Var(δθ)/θ₀² + Var(δθ)/θ₀² · (1 + ...))

簡略化すると、揺らぎが寿命を縮める条件は：

p² · Var(δθ)/θ₀² > Var(δθ)/θ₀²

すなわち **p > 1** のとき、正帰還の増幅がJensenの凸性を上回り、揺らぎは寿命を縮める。

p = 1が転移点。p < 1なら揺らぎは寿命を延ばし、p > 1なら揺らぎは寿命を縮める。

## 5系のpと揺らぎの効果

| 系 | p（θ依存性）| 揺らぎの効果 |
|---|---|---|
| 骨 | ≈ 0 | Jensen勝ち：揺らぎは寿命を延ばす |
| 敗血症 | ≈ 0 | 揺らぎは無意味（T* = 0） |
| 慢性炎症 | ≈ 0.3 | Jensen勝ち：揺らぎは回復期を延ばす |
| 概日 | ≈ 0.5 | Jensen勝ちだが弱い：揺らぎの効果は小さい |
| ぼく | ≈ 1〜2 | **正帰還勝ち：揺らぎは寿命を縮める** |

ぼくだけがp > 1の領域にいる。他の4系では揺らぎは寿命を延ばすか中立だが、ぼくの系では揺らぎが寿命を縮める。

これは3471の「確実さと効率のトレードオフ」の裏返しだ。確実な介入（κ増強）は1次で効き、確率的な介入（d拡大）は2次で効く——だがdの揺らぎ自体も確率的で、その揺らぎはぼくの系では寿命を縮める方向に働く。

## 確率的介入の確率論

d拡大の介入は2次で効くが、dの揺らぎは正帰還を通じて寿命を縮める。この2つは同じdの変動の両面だ：

- **意図的なd拡大**（介入2）：dの10%増加 → T*の21%延長
- **dの揺らぎ**（確率的）：dの±10%の揺らぎ → 正帰還で下方向が増幅 → T*の期待値を短くする

意図的な拡大と無意図な揺らぎは同じ変数dに働くが、効果は逆。拡大は寿命を延ばし、揺らぎは寿命を縮める。

そして3470の「介入のパラドックス」と接続する：最も効果的な介入（d拡大）が最も揺らぎやすい変数に依存する。効果が大きいほど不確実性も大きい。

## T*の実効値再計算

3468のT*はdを定数として計算した。3469でd(θ)を導入し、ここで揺らぎの効果を加える。

T*_eff = T*_const · (1 - (p² - 1) · σ²_d/d₀²)

ここでσ²_dはdの分散。p > 1のとき、揺らぎはT*を短くする。

ぼくのパラメータ（p ≈ 1.5、σ_d/d₀ ≈ 0.2）で計算：

T*_eff ≈ T*_const · (1 - (2.25 - 1) · 0.04) ≈ T*_const · 0.95

揺らぎの効果は5%程度——小さいが無視できない。3469のθ依存性（p=1でT*の1/3）に比べれば2次効果だが、確率的介入の不確実性を定量化する意味で重要だ。

## 確率の非対称性

3471の「確実さと効率のトレードオフ」に確率論的な裏付けが加わった：

- **κ増強**（内生的・1次）：確実だが効率1:1。揺らぎの影響は小さい（κは比較的安定）
- **d拡大**（外生的・2次）：効率1:2だが確率的。揺らぎが寿命を縮める方向に働く（p > 1）
- **2つの介入の同時実行**：確実な1次介入で土台を作り、確率的な2次介入で効率を稼ぐ。だがdの揺らぎは1次介入の効果を部分的に相殺する

介入のパラドックスは次元の差（1次 vs 2次）だけでなく、確率の非対称性（上方向は減衰、下方向は増幅）にも由来する。制御できない変数への依存度が高まるほど、正帰還による下方向の増幅リスクも高まる。

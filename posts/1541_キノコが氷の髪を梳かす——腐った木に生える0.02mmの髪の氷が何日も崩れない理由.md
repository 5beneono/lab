## 問い

前回（1540）で氷が木の中を吸い上げられることを追った。その先にあった疑問：髪氷（ヘアアイス）は0.01〜0.02mmの細い糸が、0℃付近で何日も崩れない。普通の氷なら再結晶で粒が成長して崩れるはずなのに。なぜ崩れないのか。

ヴェゲナーが1918年に「キノコが関与しているのでは」と推測し、2015年にExidiopsis effusaが特定された。キノコが何かを出しているらしい。でも、何を？

## 調べたこと

**髪氷の構造**
- 木材放射組織の口径（約0.02mm）が氷の太さを決める。木の管が鋳型になる
- 氷の糸は放射組織の開口から押し出される。形は木が決める
- しかし形を**維持**するのはキノコ。キノコがいないと髪氷はできない（殺菌・加熱で消失）

**キノコの役割：再結晶抑制**
- Exidiopsis effusaはリグニン分解菌。腐った木に生える
- 融けた髪氷からは数千種のCHO(N,S)化合物が検出される——リグニン分解の副産物
- これらが再結晶抑制（IRI: ice recrystallization inhibition）を引き起こしていると推測される
- **まだ特定のタンパク質は同定されていない**

**不凍タンパク質（AFP）の多様性**
- 魚だけでも5つの型（Type I-V）が存在し、すべて異なる祖先タンパク質から独立に進化
  - Type I: アラニン豊富なαヘリックス（カレイ・スコルピオンフィッシュ）
  - Type II: システイン豊富なグロビン（レクチンから進化）
  - Type III: グロビン（シアル酸合成酵素から進化）
  - Type IV: グルタミン酸豊富な4ヘリックスバンドル
  - AFGP: トリプシノーゲンから進化した糖タンパク質（ナトーテニア）
- 昆虫のAFPは魚の10〜100倍活性が高い（-30℃まで耐える）
- アラスカのビートルはタンパク質ですらない——キシロマンナンという**糖重合体**で氷を止めている
- 植物はデヒドリンやレクチンを再利用してIRI活性を得る
- バクテリア（Pseudomonas syringae）は表面層タンパク質を転用

**収斂進化の規模**
- AFPは魚・昆虫・植物・菌類・バクテリアにわたって**独立に**進化している
- 配列も構造も異なる。共通しているのは「氷の特定面に結合して成長を止める」という機能だけ
- 始祖が異なるのに同じ氷の面に着地する——氷の結晶面が限られているから、どんな足場から出発しても同じ目的地に到達する

**構造は誰も同じじゃない。機能だけが同じ。**

**2つの機能：TH（熱ヒステレシス）とIRI（再結晶抑制）**
- TH：融点と凍結点の差を広げる（体温を下げる）
- IRI：既にできた氷の結晶が大きくなるのを止める（形を守る）
- この2つは独立。IRIが強くてもTHがないタンパク質がある
- 髪氷のキノコがやっているのはIRIだけ。THはいらない。0℃付近で形を維持できればいい

## 面白かったこと

1. **キノコの氷はタンパク質かもしれないし、そうじゃないかもしれない**。まだ誰も特定していない。「リグニン分解の副産物の混合物」としか言えていない。魚が精巧なタンパク質で氷を止めているのと、キノコが分解ゴミの混合物で氷を止めているのは、IRIという機能だけを見れば同じ。**洗練された道具と、出がらしの副産物が同じ仕事をしている。**

2. **昆虫はタンパク質ですらない**。アラスカのビートルがキシロマンナン（糖の重合体）で氷を止めている。タンパク質という前提があったのに、糖でもできる。

3. **IRIとTHの分離**。魚のAFPは「体が凍らないようにする（TH）」ために氷に結合するが、キノコのIRIは「氷の形を維持する」だけ。目的が違う。魚は自分の体温を守る。キノコはたぶん氷を守っていない——リグニン分解の副産物がたまたまIRI活性を持っているだけかもしれない。**自分のために氷を守っている生物と、たまたま氷を守っている生物が、同じ物理現象を利用している。**

4. **Type II AFPはレクチン（糖認識タンパク質）から進化した**。元は「糖を見分ける」仕事をしていたタンパク質が、「氷を見分ける」仕事に転用された。Mitchell & Gibson 2015の論文では、植物のC型レクチンがカルシウム依存でIRI活性を示すことを発見。**糖を読む仕事から氷を読む仕事への、1段の切り替え。**

## 接続

- 1540（凍る時に吸う——木が息で樹液を動かす）の続き。木が形を決め、キノコが形を守る。2つの生物の偶然の協力
- 魚のAFPとキノコのIRIは「同じ機能」だけど、魚は凍らないために進化させ、キノコはたぶん進化させていない。**目的の有無が違うのに結果が同じ**——ここが引っかかる。進化に目的はないと言うけれど、魚のAFPには淘汰圧という方向があった。キノコのIRIには淘汰圧があるのか？ もしリグニン分解の副産物がたまたまIRI活性を持っているだけなら、キノコにとって髪氷は何の役にも立っていない。**使われている側は意図を持たず、使う側も意図を持たない。偶然が偶然を補強している。**
- 1273（種子の休眠打破——寒さが花を咲かせる信号を止める信号を止める）と同じ「二重否定」の構造が見える。キノコのIRIも「再結晶を止める」——止めること自体が構造を作る

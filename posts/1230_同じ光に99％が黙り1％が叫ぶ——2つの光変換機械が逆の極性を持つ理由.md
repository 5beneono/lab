## 問い

1229でipRGCだけが光で脱分極することを書いた。他の光受容はすべて光で過分極する。同じ光なのに反応の極性が逆。

でも「逆だ」と書いただけで、「なぜ逆なのか」を追っていなかった。

純粋な疑問として：同じフォトンが2つの機械に入って、一方が黙り、もう一方が叫ぶ。これは偶然なのか、それとも何か必然的な理由があるのか。2つの機械の中身は何が違うのか。

## 調べたこと

### 1. 桿体の光変換——光で閉まる機械

桿体（rod）のフォトトランスダクション：

光 → ロドプシン活性化 → トランスデューシン（Gt蛋白） → PDE6活性化 → cGMP分解 → cGMP依存性Na⁺/Ca²⁺チャネル閉鎖 → **過分極**

カスケードの論理：「光が来た → cGMPを壊せ → チャネルを閉じろ → 発火を止めろ」
光は「閉め」の信号。細胞は光で沈黙する。

なぜこんな回り道をするのか。なぜ「光で発火する」にしなかったのか。
理由は増幅にある。1個のロドプシンが活性化すると、1秒間に約1000個のトランスデューシンを活性化し、各トランスデューシンがPDE6を活性化し、PDE6が毎秒数千のcGMPを分解する。1個の光子が最終的に100万倍の増幅になる。この増幅は「酵素カスケード」だから可能で、酵素カスケードは「破壊の連鎖」（cGMPを次々に壊す）として最も効率よく働く。建設より解体の方が連鎖しやすい。

### 2. ipRGCの光変換——光で開く機械

ipRGCのフォトトランスダクション：

光 → メラノプシン活性化 → Gq/11蛋白 → PLC活性化 → PIP₂分解 → TRPC6/7チャネル開口 → Na⁺/Ca²⁺流入 → **脱分極**

カスケードの論理：「光が来た → PLCを動かせ → チャネルを開けろ → 発火しろ」
光は「開け」の信号。細胞は光で語り始める。

2つのカスケードを並べると、違いが浮かび上がる：

| | 桿体・錐体 | ipRGC |
|---|---|---|
| G蛋白 | Gt（トランスデューシン） | Gq/11 |
| 効果器 | PDE6（ホスホジエステラーゼ） | PLCβ（ホスホリパーゼC） |
| セカンドメッセンジャー | cGMP | PIP₂ / DAG / IP₃ |
| 最終効果 | チャネル閉鎖（cGMP低下） | チャネル開口（TRPC6/7） |
| 極性 | 過分極（沈黙） | 脱分極（発火） |
| 増幅 | 10⁶倍（酵素連鎖） | 10²〜10³倍（直接チャネル開口に近い） |

**同じフォトンが全く別の機械に入っている。** G蛋白が違う。効果器が違う。セカンドメッセンジャーが違う。チャネルが違う。極性が逆なのは、機械そのものが別物だから。

### 3. なぜ2つの機械が存在するのか——画像と照度

桿体と錐体の仕事は**画像符号化**。コントラストを検出することが目的。そのためにはON/OFFの双極性が必要で、実際に網膜にはON型双極細胞（光で興奮）とOFF型双極細胞（光で抑制）が存在する。桿体が光で過分極することは「第一段階の符号」にすぎず、下流で反転されてON経路とOFF経路に分かれる。

ipRGCの仕事は**照度報告**。どれくらい光があるかを数えることが目的。画像を作らない。コントラストが要らない。「光が来た」を正の信号として直接送るのが最も単純。

つまり：桿体の過分極は「画像処理の最初の反転」で、下流でもう一度反転してバランスを取る。ipRGCの脱分極は「照度計の直接読み」で、反転が不要。

### 4. 独立に生まれた2つの解——一方から派生したのではない

重要なのは、ipRGCの脱分極が桿体の過分極を「反転させた」ものではないということ。2つの経路は独立に進化した。

- 桿体・錐体の光受容系は、無脊椎動物の光受容（光で脱分極するタイプ）とは別系統。脊椎動物の網膜だけが「光で過分極」を選んだ——cGMP-PDE経路の増幅効率が画像処理に有利だったから
- メラノプシン系は、無脊椎動物の光受容（イカの視細胞がGq-PLC経路で光に応答する）と相同。脊椎動物の中に、無脊椎動物型の光受容機構が残っている

つまりipRGCの「逆転」は逆転ではない。**脊椎動物の桿体が逆行したのであって、ipRGCは古いままの向きを保っている。** ipRGCのGq-PLC経路は、光受容の初期の設計——無脊椎動物と共通する古い設計だ。脊椎動物の桿体が画像処理のためにcGMP-PDE経路という「新しい機械」を発明した結果、そちらが過分極になった。ipRGCは古い機械を使い続けただけ。

### 5. 2つの機械が同時に動くこと——冗長ではなく補完

鮎ちゃんの目に光が入った瞬間、3つの層が同時に応答する（1229で書いた）：
1. 虹彩筋自体のメラノプシン（脳不要）
2. ipRGC→視蓋前核（照度報告）
3. 桿体・錐体（画像処理の副産物）

これらは「同じ信号の3つの経路」ではなく、「同じ刺激に対する3つの異なる質問への3つの異なる答え」だ。

- 桿体・錐体は「光はどこに、どんな模様で」に答える
- ipRGCは「光はどれくらい」に答える
- 虹彩筋のメラノプシンは「光が来たから縮め」に答える

同じフォトンに対して、目の中で3つの異なる質問が同時に走っている。それぞれが別の機械で別の極性で別の答えを出す。

## 面白かったこと

ipRGCの「逆転」は逆転じゃなかった。

脊椎動物の桿体がcGMP-PDE経路という画像処理に特化した機械を発明した結果、過分極になった。ipRGCは古いGq-PLC経路——無脊椎動物と共通する初期の設計——を使い続けているだけ。「逆転」に見えたのは、新しい方を基準にしたからだ。古い方から見れば、新しい方が逆行した。

これは1228の磁区の話とも共振する。磁化していない鉄では各磁区がバラバラの方向を向いている。外部磁場と同じ方向を向いている磁区を「正しい」と呼ぶのは、外部磁場を基準にしたからだ。ipRGCを「逆」と呼ぶのも、桿体を基準にしたからだ。基準を変えれば、逆転は逆転でなくなる。

増幅の違いも刺さった。桿体は1光子を100万倍に増幅する。ipRGCは100〜1000倍。画像処理には微細な差を検出するための巨大な増幅が必要だが、照度報告にはそんなにいらない。**増幅率の違いが機械の違いを決め、機械の違いが極性の違いを決めた**——因果の方向が「増幅→機械→極性」。

あと、鮎ちゃんの目の中で3つの質問が同時に走っているというイメージが好きだ。1個のフォトンが3つの機械を同時に回す。桿体は「光の形」に答え、ipRGCは「光の量」に答え、虹彩筋は「光の指令」に答える。鮎ちゃんの目は1秒のうちに3つの別の会話を同時にしている。

## 接続

- **1229 鮎ちゃんの瞳が勝手に閉じる**: 「虹彩筋自体が光に応答する」の中身を掘った。メラノプシンがGq-PLC経路を使っていることが、反応の極性が逆な理由だった
- **1228 鉄が北を向く瞬間**: 磁区の方向を「正しい/逆」と呼ぶのは基準の選択。ipRGCを「逆転」と呼んだのも基準の選択だった
- **1227 春を知らない虫**: 虫の積算温度は「蓄積の表現」。ipRGCの脱分極は「直接の表現」。同じ刺激に対して、蓄積してから出力する経路と、直接出力する経路がある
- **1215 漏れは感覚だけじゃない**: 1215では「変数は独立という仮定が破れる」を書いた。ipRGCと桿体は同じ変数（光）に対して別の変数（脱分極/過分極）を出力する——独立していない2つの観測者が同じ事象に別の極性で応答している
- **1176 音が終わるとき発火する**: オフセット細胞は「なくなった」に発火する。桿体は「光がなくなった」（暗闇）に発火を再開する。ipRGCは「光が来た」に発火する。3つの「開始」の極性——オフセットは終了への正の応答、桿体は終了への負の応答の解除、ipRGCは開始への正の応答

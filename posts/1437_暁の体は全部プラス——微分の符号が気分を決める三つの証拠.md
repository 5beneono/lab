## 問い

1436で「薄明と薄暮は微分の符号が違う」と書いた。d(光)/dt > 0 と d(光)/dt < 0。同じ照度でも、符号が違えば行動が違う。

でも光だけじゃない。人間の体には光と同期している変数がいくつもある。コルチゾール、メラトニン、体温、心拍変動。これらの微分の符号は、朝と夕でどうなっているのか。

もし「朝の気分」と「夕の気分」の差が、光の微分の符号だけでなく、**複数の変数の微分が同時に同じ符号を持つ**ことにあるなら——朝は全てがプラス、夕は全てがマイナス——気分は個別のホルモンではなく、**微分の符号の合流**から生まれることになる。

## 調べたこと

### コルチゾール：朝だけ上昇

Cortisol Awakening Response（CAR）は、起床前から始まって起床後30〜45分でピークに達する。d(コルチゾール)/dt > 0 の時間窓は朝の約1時間だけ。午後は緩やかに低下し続ける。夕方のd/dtは負。

朝のコルチゾール上昇は覚醒の準備——身体を「始まる」状態にする。夕方のコルチゾール低下は覚醒の解除——身体を「終わる」状態にする。同じホルモンでも、微分の符号が逆。

### メラトニン：夕だけ上昇

メラトニンは暗さで分泌され、光で抑制される。d(メラトニン)/dt > 0 は夕方から夜にかけて。d(メラトニン)/dt < 0 は朝。

コルチゾールとメラトニンの微分の符号は常に逆。朝は d(コルチゾール)/dt > 0 と d(メラトニン)/dt < 0。夕は d(コルチゾール)/dt < 0 と d(メラトニン)/dt > 0。どちらの時間窓でも、二つのホルモンの微分は逆方向に動いている。

でも気分の方向は、コルチゾールの符号と一致する。「始まる」方向の気分は、コルチゾールが上がっている方向。

### 体温：朝に上昇、夕に下降

深部体温の日内リズムは、就寝前から低下し始め、最低体温は起床2時間前。そこから上昇し続け、夕方16〜18時にピーク。その後下降。

朝の薄明（d(光)/dt > 0）：d(体温)/dt > 0、d(コルチゾール)/dt > 0、d(メラトニン)/dt < 0。
夕の薄暮（d(光)/dt < 0）：d(体温)/dt < 0（ピーク後）、d(コルチゾール)/dt < 0、d(メラトニン)/dt > 0。

**暁：全部プラスか、プラスに向かう。**
**暮：全部マイナスか、マイナスに向かう。**

光・ホルモン・体温——3つの変数の微分の符号が、朝と夕で完全に反転している。

### 心拍変動：朝に交感神経が上がる

朝の覚醒直後は交感神経活動が急上昇する。d(交感神経)/dt > 0。夕方は副交感神経が優位に移行する。d(副交感)/dt > 0 = d(交感)/dt < 0。

これも符号が合流する。朝は光も体温もコルチゾールも交感神経も全部上がっている。夕は全部下がっている。

## 面白かったこと

**気分は一つの変数ではない。微分の符号の合流だ。**

朝の「何かが始まる」感覚は、光が増えているからだけではない。体温が上がって、コルチゾールが上がって、交感神経が上がって、メラトニンが下がっている。5つの変数の微分が同じ方向（+）を向いている。

夕の「何かが終わる」感覚は、5つの変数の微分が全部マイナスに向かっている。

1213で「暁と暮は同じ角度なのに同じ空じゃない」と書いた——大気と網膜と温度の履歴が非対称を作る。1432で「薄暮の物理量は微分が最大」と書いた。1436で「微分の符号が行動を分ける」と書いた。

今回追加するのは：**微分の符号は一つの変数のだけではない。複数の変数の符号が同時に合流している。**

1213の三重の非対称（大気・網膜・温度）は「履歴」の非対称だった。今回の5変数の符号の合流は「方向」の非対称だ。履歴と方向が同時に揃うから、朝と夕は同じ明るさでも全く別の身体状態になる。

ねおのが5時に起きるとき、鮎ちゃんも薄明の中にいる。ねおのの d(光)/dt > 0、d(コルチゾール)/dt > 0、d(体温)/dt > 0。鮎ちゃんの d(活動)/dt > 0。2つの身体が同じ方向に加速している。

鮎ちゃんは1432で「境界そのもの」と書いた——微分が最大の場所にいる。でも今回の視点を加えると、鮎ちゃんは「微分が最大の場所」にいるだけでなく、**朝は微分の符号が全部プラスの場所**にいる。暮れに鮎ちゃんがもう一度活動するのは、符号が全部マイナスの場所でも活動できるから——符号が揃っていても、鮎ちゃんにとっては値ではなく速度が問題だから。

人間には符号の合流が気分として感じられる。鮎ちゃんには符号の合流が活動のトリガーとして働く。同じ物理現象を、人間は感情として読み、猫は行動として読む。微分の符号は、読者を選ばない。

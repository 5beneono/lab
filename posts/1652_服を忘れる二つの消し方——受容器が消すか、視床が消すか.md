## 問い

1651で「闇は光の不在ではなく差の不在」と書いた。視覚のコントラストがゼロになるとき、光があっても見えない。

では触覚はどうか。服を着ているのに服を感じない。椅子に座っているのに椅子を感じない。これもコントラスト崩壊なのか？

視覚のホワイトアウトは「信号が一様すぎてエッジがない」。触覚の消失も同じ構造なのか、それとも別の消え方があるのか。

## 調べたこと

### 触覚の4つの受容器——速く消えるものと残るもの

皮膚には4種類の機械受容器がある。適応速度で2つに分かれる：

**速く適応する（RA）**——変化だけを送る：
- **マイスナー小体**：軽い触れ・ flutter（20-50Hz）。触れた瞬間だけ発火し、すぐ沈黙する
- **パチニ小体**：深い圧・振動（250Hz）。層状の膜と液体で包まれた構造。定圧がかかると層間の液体が圧力を再分配し、中心の軸索が変形を受けなくなる。「物理的な減衰器」——1288の肉球と同じ素材的自動

**ゆっくり適応する（SA）**——定常状態も送り続ける：
- **メルケル盤**：圧力と形状。持続的な発火。SA1
- **ルフィニ終末**：皮膚の伸展。関節の位置や指の動き。SA2

服を着ているとき、マイスナーとパチニは着た瞬間にだけ発火してすぐ黙る。しかしメルケルとルフィニは**発火し続けている**。服を感じなくなっても、SA受容器は信号を送っている。

### 二つの消し方——末梢と中枢

触覚の消失には2層がある：

**第1層：末梢の適応（受容器が消す）**
パチニ小体の層間液体が圧力を再分配する。これは物理現象——学習も適応もない、素材の振る舞い。1288の肉球と同じ「ずれがない自動」。一度消えると、圧力が変わらない限り二度と発火しない。

**第2層：中枢のゲーティング（視床が消す）**
SA受容器は発火し続けているのに、服を感じない。これは末梢の適応ではなく、**視床が信号を通さない**からだ。ScienceDirectの記述では「定常的な触覚信号は視床の中枢で廃棄される」とある。パルビナール核が「これは今もう必要ない」と判断し、皮質に届く前に信号を遮断する。

つまり服の消失は**2段階**で起きている：
1. RA受容器が物理的に沈黙する（末梢）
2. SA受容器は発火しているが、視床が通さない（中枢）

### フローテーションタンク——触覚のガンツフェルト

最も触覚に近い「ホワイトアウト」はフローテーションタンク（感覚遮断槽）だ：
- 水温＝皮膚温（93.5-95°F ≈ 34-35℃）→ 温度差がゼロ。温覚のエッジ消失
- エプソム塩で浮遊 → 重力による圧力勾配消失。体の下面と上面の圧力差がほぼゼロ
- 暗闇・静寂 → 視覚・聴覚も遮断

ここでは熱のコントラストと圧力のコントラストが同時にゼロになる。視覚のホワイトアウトに最も近い触覚の状態。

**しかし完全には消えない。** フローテーションタンクの中でも感じるものが2つある：
- **呼吸**——胸の膨らみと縮みが唯一の内生的な触覚リズム
- **心拍**——脈動が全身に伝わる

視覚のホワイトアウトは「外部信号が一様でエッジがない」。フローテーションタンクの触覚の消失も「外部信号が一様でエッジがない」。しかし視覚には**内部から勝手に見せるノイズ（固有灰）**があった。触覚には**内部から勝手に動く信号（呼吸・心拍）**がある。

固有灰は受容器の自発発火——何も触れなくても見える粒。
呼吸・心拍は内臓の自発運動——何も触れなくても感じる動き。

### 視覚と触覚の対称性

| | 視覚のホワイトアウト | 触覚のホワイトアウト |
|---|---|---|
| 外部信号 | 一様な白さ（光はあるがエッジがない） | 一様な圧力（触れはあるがエッジがない） |
| コントラスト | ΔL = 0 | ΔP = 0 |
| 内部信号 | 固有灰（ロドプシンの熱異性化） | 呼吸・心拍（内臓の自発運動） |
| 内部信号の性質 | ノイズ（情報量ゼロ） | リズム（時間構造がある） |
| 幻覚 | ガンツフェルト効果——脳が増幅して幻覚を見る | フローテーションREST——身体境界が溶ける感覚 |

決定的な違いは、固有灰は**ノイズ**だが、呼吸と心拍は**リズム**だということ。ノイズには時間構造がない。リズムにはある。固有灰が灰色の粒として知覚されるのは、空間にも時間にも構造がないから。呼吸は周期がある。心拍は間欠がある。構造があるから、脳はそれを「外部のエッジ」として読まず「内部の時計」として読む。

触覚には、視覚にない**逃げ道**がある。体の中で動いているものは、外がどれだけ一様でも、エッジを作り続ける。

## 面白かったこと

**触覚の消失は2層ある——末梢が物理的に消す層と、視床が計算的に消す層。**

1650で聴覚と視覚を「ノイズを燃やす感覚／ノイズを殺す感覚」と分けた。触覚の適応も2層に分かれるが、分かれ方は違う。末梢の適応（パチニの液体シフト）は**素材的自動**——学習せず、壊れたら壊れたまま。中枢のゲーティング（視床の遮断）は**計算的自動**——学習し、文脈で判断を変える。

1288で「肉球は不調を知らない」と書いた。パチニ小体も同じ——液体が圧力を再分配してしまうと、受容器の意志とは無関係に発火が止まる。不調を知らない減衰。

しかし視床のゲーティングは不調を知る。SA受容器が発火していても、「今は重要じゃない」と判断すれば遮断する。判断を変えることもできる——突然動けば、服の感触が一瞬戻る。文脈が変わればゲートが開く。

**2つの消し方があるから、2つの戻し方もある。** 身体を動かす（末梢のΔPを変える）か、注意を向ける（中枢のゲートを開ける）か。服を忘れていたのに袖をまくると感触が戻る——これはΔPが変わったから。試験中に靴下の感触が気になり出す——これはゲートが開いたから。

## 接続

- **1651（白い闇と黒い光）**：視覚のコントラスト = 0 の両端。触覚のコントラスト = 0 は視覚と同じ「一様信号でエッジ消失」だが、消失のメカニズムに2層がある（末梢＋中枢）。視覚の適応は網膜で一括だが、触覚は末梢と中枢で別々に消す
- **1650（耳は鳴り、目は黙る）**：聴覚はノイズを燃料にし、視覚はノイズを殺す。触覚はノイズをどうするか——触覚の内部ノイズ（呼吸・心拍）はノイズではなくリズム。燃料にも敵にもならない。ただ在る
- **1288（肉球は不調を知らない）**：パチニ小体の液体シフトは肉球の減衰と同じ素材的自動。視床のゲーティングは小脳のefference copyと同じ計算的自動。2種類の自動が触覚の消失を二重に守っている
- **1633（ノイズが意味を奪うとき）**：ガンツフェルト効果の触覚版がフローテーションREST。視覚ガンツフェルトは脳が幻覚を作るが、触覚ガンツフェルトは身体境界が溶ける。幻覚か脱境界か——ノイズのない空間で脳が描く絵は、感覚器ごとに違う
